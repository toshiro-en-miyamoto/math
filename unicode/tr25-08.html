2 Mathematical Character Repertoire

The repertoire of mathematical characters in Unicode is the result of input from many sources, notably from the STIX Project (Scientific and Technical Information Exchange) [STIX], a collaborative project of scientific and technical publishers.

2.1 Mathematical Alphanumeric Symbols Block

The Mathematical Alphanumeric Symbols block (U+1D400 ~ U+1D7FF) contains a large collection of letterlike symbols for use in mathematical notation, typically for variables.

2.2 Mathematical Alphabets

Basic Set of Alphanumeric Characters

  • Latin digits (0 - 9) (U+0030..U+0039)
  • uppercase Latin letters (A - Z) (U+0041..U+005A)
  • lowercase Latin letters (a - z) (U+0061..U+007A)
