Polynomial in Linear Algebra [線形代数における多項式]

Vector Spaces

Polynomial [多項式]: A polynomial with coefficients [係数] from a field F is an expression [式] of the form

f(x) = an⁢xn + an-1⁢xn-1 + ⋯ + a1⁢x + a0

Degree [次数]: The degree of a polynomial is defined to the largest exponent of x with a non-zero coefficient.

Zero polynomial: A polynomial f(x) is call the zero polynomial if ai = 0 for 0 ≤ i ≤ n, that is f(x) = 0. Its degree is defined to be −1.

Equality of polynomials: Two polynomials,

f(x) = an⁢xn + an-1⁢xn-1 + ⋯ + a1⁢x + a0
g(x) = bm⁢xm + bm-1⁢xm-1 + ⋯ + b1⁢x + b0
are called equal if m = n and ai = bi for i = 0, 1, …, n.

Notation (P(F): a set of polynomials): A set of all polynomials with coefficients from a field F, denoted P(F), is a vector space with the two operations — suppose m ≤ n, and bm+1 = bm+2 = ⋯ = bn = 0, then

f(x)+g(x) = (an+bn)⁢xn + (an-1+bn-1)⁢xn-1 + ⋯ + (a1+b1)⁢x + (a0+b0)
c⁢f(x) = c⁢an⁢xn + c⁢an-1⁢xn-1 + ⋯ + c⁢a1⁢x + c⁢a0

Subspaces

Notation (Pn(F): a set of all polynomials with degree ≤ n): Let n ≥ 0, and let Pn(F) consist of all polynomials in P(F) having degree ≤ n.

Proposition (Pn(F) is a subspace): Pn(F) is a subspace of P(F).

Proof: Let p1, p2 ∈ Pn(F), and let c ∈ F,

zero ∈ Pn(F) ∵ the zero polynomial has degree -1 < n
p1+p2 ∈ Pn(F) ∵ p1+p2 is another polynomial of degree ≤ n
cp1 ∈ Pn(F) ∵ cp1 is another polynomial of degree ≤ n
Hence by theorem (subspace conditions), Pn(F) is a subspace of P(F). ▮

Linear Combinations and Systems of Linear Equations

Systems of linear equations [連立一次方程式 | 線型方程式系]: A system of linear equations consists of two or more linear equations made up of two or more variables such that all equations in the system are considered simultaneously.

To find the unique solution [解] to a system of linear equations, we must find a numerical value for each variable in the system that will satisfy all equations in the system at the same time. In order for a linear system to have a unique solution, there must be at least as many equations as there are variables. Even so, this does not guarantee a unique solution. ( Systems of Linear Equations, LibreTexts Mathematics )

Example (1.4.2: linear combination of polynomials): Consider linear combinations of two polynomials

x3−2x2−5x−3
3x3−5x2−4x−9
Case 1: 2x3−2x2+12x−6 is a linear combination of the polynomials in P3(ℝ)

2x3−2x2+12x−6 = a(x3−2x2−5x−3) + b(3x3−5x2−4x−9)
= (a+3b)x3 + (−2a−5b)x2 + (−5a−4b)x + (−3a−9b)
Thus we are led to the following system of linear equations, adding appropriate multiples of the first equation to the others in order to eliminate a, and adding the appropriate multiples of the second equation to the others yields a and b:
a+3b =2 ⟶ a+3b =2 ⟶ a =−4
−2a−5b =−2 b =2 b =2
−5a−4b =12 11b =22 0 =0
−3a−9b =−6 0b =0 0 =0
Hence 2x3−2x2+12x−6 is a linear combinations of the two polynomials:
2x3−2x2+12x−6 = −4(x3−2x2−5x−3) + 2(3x3−5x2−4x−9)

Case 2: 3x3−2x2+7x+8 is not a linear combination of the polynomials

3x3−2x2+7x+8 = a(x3−2x2−5x−3) + b(3x3−5x2−4x−9)
Using the preceding technique, we obtain a system of linear equations, and eliminating a as before yields
a+3b =3 ⟶ a+3b =3
−2a−5b =−2 b =4
−5a−4b =7 11b =22
−3a−9b =8 0 =17
But the presence of the inconsistent equation 0 = 17 indicates that the system of linear equations has no solution. Hence 3x3−2x2+7x+8 is not a linear combination of the two polynomials.
