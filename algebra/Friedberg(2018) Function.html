Function [関数]

Sets (Appendix A)

Definition (empty set): The empty set, denoted ∅, is a set containing no elements. ∅ is a subset of every set.

Functions (Appendix B)

Definition (function): If A and B are sets, then a function f from A to B, denoted

f : A ⟼ B is a rule that
associates each element x in A
with the one and only element in B, denoted f(x).

Equality of function: Two functions

f : A ⟼ B and g : A ⟼ B
are equal if f(x) = g(x) for all x ∈ A.

domain A of f : A ⟼ B
range the set { f(x) : x ∈ A }
one-to-one function if f(x) = f(y) implies x = y
onto functions the range { f(x) : x ∈ A } = B
functions need not be one-to-one
with x ≠ y, it is allowed that f(x) = f(y)
functions need not be onto
in general, the range { f(x) : x ∈ A } ⊆ B

Definition (composite): Let A, B, and C be sets and

f : A ⟼ B and g : B ⟼ C
be functions. Then a function composite of g and f is denoted
g∘f : A ⟼ C ⋯ (g∘f)(x) = g(f(x)) for all x ∈ A

For example, let A = B = C = ℝ,

f(x) = sin x and g(x) = x2+3
then
g∘f : ℝ ⟼ ℝ ⋯ (g∘f)(x) = (g(f(x))) = (sin x)2+3
f∘g : ℝ ⟼ ℝ ⋯ (f∘g)(x) = (f(g(x))) = sin(x2+3)

Proposition (composite associativity): Functional composition is associative. Let f, g, and h be functions, then

h∘(g∘f) = (h∘g)∘f
Proof: We prove that both sides are equal for every input x.
(h∘(g∘f))(x) = h((g∘f)(x)) ⋯ composite of h and g∘f
= h(g(f(x))) ⋯ composite of g and f
= (h∘g)(f(x)) ⋯ h∘g that takes f(x) as input
= ((h∘g)∘f)(x) ⋯ composite of h∘g and f

Notation (𝓕(S, F): a set of functions) : Let S be any nonempty set and F be any field, and let 𝓕(S, F) denote the set of all functions from S to F.

𝓕(S, F) = { f : S ⟼ F }
Two functions f and g in 𝓕(S, F) are called equal if
f(s) = g(s) for each s ∈ S

Proposition (example 1.2.3: function sets as vector space): The set 𝓕(S, F) is a vector space with the operations of addition and scalar multiplication, for f, g ∈ 𝓕(S, F) and c ∈ F by, for each s ∈ S

(f+g)(s) = f(s)+g(s) ⋯ (FA)
(cf)(s) = cf(s) ⋯ (FM)
Proof: We show that functions f, g, h ∈ 𝓕(S, F) satisfy vector space axioms. Let a, b ∈ F.
vector addition scalar multiplication
closed f+g ∈ 𝓕 cf ∈ 𝓕
commutativity f+g = g+f
associativity (f+g)+h = f+(g+h) (ab)f = a(bf)
existence of identity ∃0 ∈ 𝓕, f+0 = f 1f = f
existence of inverse ∃(f−1) ∈ 𝓕, f+(f−1) = 0
distributivity a(f+g) = af+ag (a+b)f = af+bf

Closed vector addition:

(f+g)(s) = f(s)+g(s) by definition (FA)
∈ F ∵ f(s), g(s) ∈ F
hence
f+g ∈ 𝓕 ∵ f+g : S ⟼ F
Closed scalar multiplication:
(cf)(s) = cf(s) by definition (FM)
∈ F ∵ c, f(s) ∈ F
hence
cf ∈ 𝓕 ∵ cf : S ⟼ F
Commutativity of vector addition:
(f+g)(s) = f(s)+g(s) by definition (FA)
= g(s)+f(s) ∵ f(s), g(s) ∈ F
= (g+f)(s) by definition (FA)
Associativity of vector addition:
((f+g)+h)(s) = (f+g)(s)+h(s) by definition (FA)
= (f(s)+g(s))+h(s) by definition (FA)
= f(s)+(g(s)+h(s)) ∵ f(s), g(s), h(s) ∈ F
= f(s)+(g+h)(s) by definition (FA)
= (f+(g+h))(s) by definition (FA)
Associativity of scalar multiplication:
(ab)f(s) = a(bf(s)) ∵ a, b, f(s) ∈ F
Existence of additive identity:

Since F is a field, F contains 0, the zero vector. Hence 𝓕 should contain a function that maps an element s ∈ S to 0. Let 0(s) denote a function such that 0(s) = 0 for all s ∈ S.

(f+0)(s) = f(s)+0(s) by definition (FA)
= f(s)+0 ∵ 0(s) = 0
= f(s) by definition of 0 ∈ F
Existence of scalar multiplicative identity:
(1f)(s) = 1f(s) by definition (FM)
= f(s) by definition of 1 ∈ F
Existence of additive inverse:

Since F is a field, an element f(s) ∈ F has its additive inverse in F, denoted −f(s). Hence 𝓕 should contain a function that maps an element s ∈ S to −f(s). Let f−1(s) denote the function such that f−1(s) = −f(s).

(f+(f−1))(s) = f(s)+f−1(s) by definition (FA)
= f(s)+(−f(s)) ∵ f−1(s) = −f(s)
= 0 ∵ −f(s) is an additive inverse of f(s)
Distributivity over vector addition:
(a(f+g))(s) = a(f+g)(s) by definition (FM)
= a(f(s)+g(s)) by definition (FA)
= af(s)+ag(s) ∵ a, f(s), g(s) ∈ F
Distributivity over scalar multiplication:
((a+b)f)(s) = (a+b)f(s) by definition (FM), note that (a+b) ∈ F
= af(s)+bf(s) ∵ a, b, f(s) ∈ F

Definition (continuous real-valued function): (Wikipedia)

  • A real-valued function is a function whose values are real numbers.
  • A real function is a function from real numbers to real numbers.
  • A continuous function is a function such that a small variation of the argument induces a small variation of the value of the function. This implies there are no abrupt changes in value.

Continuity: (Wikipedia)

  • Continuity is one of the core concepts of calculus and mathematical analysis, where arguments and values of functions are real and complex numbers. The concept has been generalized to functions between metric spaces and between topological spaces. The latter are the most general continuous functions, and their definition is the basis of topology.
  • A real function that is a function from real numbers to real numbers can be represented by a graph in the Cartesian plane; such a function is continuous if, roughly speaking, the graph is a single unbroken curve whose domain is the entire real line.

Notation (C(ℝ): set of CR functions): Let C(ℝ) the set of all continuous real-valued functions defined on ℝ, that is

C(ℝ) = { continuous real-valued function : ℝ ⟼ ℝ }

Proposition (example 1.3.2: C(ℝ) as subspace): C(ℝ) is a subspace of the vector space 𝓕(ℝ, ℝ).

Proof: We show that C(ℝ) meets the three conditions of the subspace.

0 ∈ C(ℝ):

The zero vector 0 in 𝓕(ℝ, ℝ) is the constant function defined by f(x) = 0 for all x ∈ ℝ. Since constant functions are continuous, 0 ∈ C(ℝ).

f+g ∈ C(ℝ):

Since the sum of two continuous functions (say f, g) is continuous, f+g ∈ C(ℝ).

xf ∈ C(ℝ):

Since the product of a real number x and a continuous function f is continuous, xf ∈ C(ℝ).

Polynomials

Polynomial [多項式]: A polynomial with coefficients [係数] from a field F is an expression [式] of the form

f(x) = an⁢xn + an-1⁢xn-1 + ⋯ + a1⁢x + a0
The degree [次数] of a polynomial is defined to the largest exponent of x with a non-zero coefficient.

Zero polynomial:

  • is a polynomial with an, an−1, …, a1, a0 = 0
  • that is f(x) = 0
  • denoted f0, that is f0(x) = 0
  • its degree is defined to be −1

Polynomials with degree zero: are written in the form f(x) = c for some nonzero scalar c.

Equality of polynomials: Two polynomials,

f(x) = an⁢xn + an-1⁢xn-1 + ⋯ + a1⁢x + a0
g(x) = bm⁢xm + bm-1⁢xm-1 + ⋯ + b1⁢x + b0
are called equal if m = n and ai = bi for i = 0, 1, …, n.

Interpretation (a polynomial as a function): With a field F containing infinitely many scalars, we usually regard

a polynomial with coefficients from F as a function: F ⟼ F

Notation (P(F): a set of polynomials): A set of all polynomials with coefficients from a field F is denoted P(F).

Proposition (example 1.2.4: P(F) is a vector space): P(F) is a vector space with the two operations — suppose m ≤ n, and bm+1 = bm+2 = ⋯ = bn = 0, then

f(x)+g(x) = (an+bn)⁢xn + (an-1+bn-1)⁢xn-1 + ⋯ + (a1+b1)⁢x + (a0+b0)
c⁢f(x) = c⁢an⁢xn + c⁢an-1⁢xn-1 + ⋯ + c⁢a1⁢x + c⁢a0
Proof: similar to the proof of Proposition (functions as a vector space).

Notation (Pn(F): a set of all polynomials with degree ≤ n): Let n ≥ 0, and let Pn(F) consist of all polynomials in P(F) having degree ≤ n.

Proposition (example 1.3.1: Pn(F) is a subspace) : Pn(F) is a subspace of P(F).

Proof: We show that Pn(F) meets the subspace conditions. Let p1, p2 ∈ Pn(F), and let c ∈ F,

f0 ∈ Pn(F) ∵ the zero polynomial f0 has degree -1 < n
p1+p2 ∈ Pn(F) ∵ p1+p2 is another polynomial of degree ≤ n
cp1 ∈ Pn(F) ∵ cp1 is another polynomial of degree ≤ n ▮

Sequences

Definition (sequence): (WikiPedia) In mathematics, a sequence is an enumerated collection of objects […]. Like a set, a sequence contains members (also called elements, or terms). The number of elements (possibly infinite) is called the length of the sequence. Unlike a set,

  • the same elements can appear multiple times at different positions in a sequence, and
  • the order does matter.
Formally, a sequence can be defined as a function
  • from natural numbers (the positions of elements in the sequence)
  • to the elements at each position.

Notation ((an): sequence): A sequence in a field F is defined by

a function σ : ℕ ⟼ F ⋯ σ(n) = an ∈ F
for n ∈ ℕ, denoted (an) n∈ℕ, that is
(an) n∈ℕ = σ(1), σ(2), …
= a1, a2, …

Proposition (example 1.2.5: sequence set as vector space): Let a set V consist of all sequences in F, then V is a vector space with the two operations, for (an) n∈ℕ, (bn) n∈ℕ ∈ V and a scalar t ∈ F

(an) n∈ℕ + (bn) n∈ℕ = (an+bn) n∈ℕ = a1+b1, a2+b2, … ⋯ (QA)
t(an) n∈ℕ = (tan) n∈ℕ = ta1, ta2, … ⋯ (QM)
Proof: We show that sequences in F satisfy vector space axioms. Let (cn) n∈ℕ ∈ V, and u ∈ F.
vector addition
closed (an) n∈ℕ + (bn) n∈ℕ ∈ V
commutativity (an) n∈ℕ + (bn) n∈ℕ = (bn) n∈ℕ + (an) n∈ℕ
associativity ((an) n∈ℕ + (bn) n∈ℕ) + (cn) n∈ℕ = (an) n∈ℕ + ((bn) n∈ℕ + (cn) n∈ℕ)
identity ∃(0n) n∈ℕ ∈ V, (an) n∈ℕ + (0n) n∈ℕ = (an) n∈ℕ
inverse ∃(−an) n∈ℕ ∈ V, (an) n∈ℕ + (−an) n∈ℕ = (0n) n∈ℕ
distributivity t((an) n∈ℕ + (bn) n∈ℕ) = t(an) n∈ℕ + t(bn) n∈ℕ
scalar multiplication
closed t(an) n∈ℕ ∈ V
associativity (tu)(an) n∈ℕ = t(uan) n∈ℕ
identity 1(an) n∈ℕ = (an) n∈ℕ
distributivity (t+u)(an) n∈ℕ = t(an) n∈ℕ + u(an) n∈ℕ

Closed vector addition:

(an) n∈ℕ + (bn) n∈ℕ = (an+bn) n∈ℕ by definition (QA)
∈ V ∵ an, bn ∈ F
Closed scalar multiplication:
t(an) n∈ℕ = (tan) n∈ℕ by definition (QM)
∈ V ∵ an, t ∈ F
Commutativity of vector addition:
(an) n∈ℕ + (bn) n∈ℕ = (an+bn) n∈ℕ by definition (QA)
= (bn+an) n∈ℕ ∵ an, bn ∈ F
= (bn) n∈ℕ + (an) n∈ℕ by definition (QA)
Associativity of vector addition:
((an) n∈ℕ + (bn) n∈ℕ) + (cn) n∈ℕ
= (an+bn) n∈ℕ + (cn) n∈ℕ by definition (QA)
= ((an+bn)+cn) n∈ℕ by definition (QA)
= (an+(bn+cn)) n∈ℕ ∵ an, bn, cn ∈ F
= (an) n∈ℕ + (bn+cn) n∈ℕ by definition (QA)
= (an) n∈ℕ + ((bn) n∈ℕ + (cn) n∈ℕ) by definition (QA)
Associativity of scalar multiplication:
(tu)(an) n∈ℕ = ((tu)an) n∈ℕ by definition (QM)
= (t(uan)) n∈ℕ ∵ t, u, an ∈ F
= t(uan) n∈ℕ by definition (QM)
Existence of vector additive identity:

Since F is a field, F contains 0, the zero vector. Hence there should exist a sequence in V that is defined by a function σ : ℕ ⟼ 0 ∈ F — for example, sin(nπ) : n ∈ ℕ ⟼ 0 ∈ ℝ, that is

sin(1π), sin(2π), … = 0, 0, …
Let (0n) n∈ℕ denote such a sequence, that is (0n) n∈ℕ = 0, 0, …
(an) n∈ℕ + (0n) n∈ℕ = (an+0) n∈ℕ by definition (QA)
= (an) n∈ℕ since an ∈ F

Existence of scalar multiplicative identity:
1(an) n∈ℕ = (1an) n∈ℕ by definition (QM)
= (an) n∈ℕ ∵ an ∈ F
Existence of vector additive inverse:

Provided a function σ(n) = an, the element an has its additive inverse in F, denoted −an, since F is a field. It implies that we can define a function ρ(n) = −an. Thus there should exist a sequence in V that is defined by the function ρ. Let denote (−an) n∈ℕ the sequence, then

(−an) n∈ℕ = ρ(1), ρ(2), …
= −a1, −a2, …
Then (−an) n∈ℕ is an additive inverse of (an) n∈ℕ;
(an) n∈ℕ + (−an) n∈ℕ = (an+(−an)) n∈ℕ by definition (QA)
= (0n) n∈ℕ −an is an additive inverse of an

Distributivity over vector addition:
t((an) n∈ℕ + (bn) n∈ℕ) = t(an+bn) n∈ℕ by definition (QA)
= (t(an+bn)) n∈ℕ by definition (QM)
= (tan+tbn) n∈ℕ ∵ an, bn, t ∈ F
= (tan) n∈ℕ + (tbn) n∈ℕ by definition (QA)
= t(an) n∈ℕ + t(an) n∈ℕ by definition (QM)
Distributivity over scalar multiplication:
(t+u)(an) n∈ℕ = ((t+u)an) n∈ℕ by definition (QM)
= (tan+uan) n∈ℕ ∵ an, t, u ∈ F
= (tan) n∈ℕ + (uan) n∈ℕ by definition (QA)
= t(an) n∈ℕ + u(an) n∈ℕ by definition (QM)

Linear Combinations and Systems of Linear Equations

Systems of linear equations [連立一次方程式 | 線型方程式系] : A system of linear equations consists of two or more linear equations made up of two or more variables such that all equations in the system are considered simultaneously.

To find the unique solution [解] to a system of linear equations, we must find a numerical value for each variable in the system that will satisfy all equations in the system at the same time. In order for a linear system to have a unique solution, there must be at least as many equations as there are variables. Even so, this does not guarantee a unique solution. ( Systems of Linear Equations, LibreTexts Mathematics)

Example (1.4.2: linear combination of polynomials) : Consider linear combinations of two polynomials

x3−2x2−5x−3
3x3−5x2−4x−9
Case 1: 2x3−2x2+12x−6 is a linear combination of the polynomials in P3(ℝ)

2x3−2x2+12x−6 = a(x3−2x2−5x−3) + b(3x3−5x2−4x−9)
= (a+3b)x3 + (−2a−5b)x2 + (−5a−4b)x + (−3a−9b)
Thus we are led to the following system of linear equations, adding appropriate multiples of the first equation to the others in order to eliminate a, and adding the appropriate multiples of the second equation to the others yields a and b:
a+3b =2 ⟶ a+3b =2 ⟶ a =−4
−2a−5b =−2 b =2 b =2
−5a−4b =12 11b =22 0 =0
−3a−9b =−6 0b =0 0 =0
Hence 2x3−2x2+12x−6 is a linear combinations of the two polynomials:
2x3−2x2+12x−6 = −4(x3−2x2−5x−3) + 2(3x3−5x2−4x−9)

Case 2: 3x3−2x2+7x+8 is not a linear combination of the polynomials

3x3−2x2+7x+8 = a(x3−2x2−5x−3) + b(3x3−5x2−4x−9)
Using the preceding technique, we obtain a system of linear equations, and eliminating a as before yields
a+3b =3 ⟶ a+3b =3
−2a−5b =−2 b =4
−5a−4b =7 11b =22
−3a−9b =8 0 =17
But the presence of the inconsistent equation 0 = 17 indicates that the system of linear equations has no solution. Hence 3x3−2x2+7x+8 is not a linear combination of the two polynomials.
