Function [関数]

Sets (Appendix A)

Definition (empty set): The empty set, denoted ∅, is a set containing no elements. ∅ is a subset of every set.

Functions (Appendix B)

Definition (function): If A and B are sets, then a function f from A to B, denoted

f : A ⟼ B is a rule that
associates each element x in A
with the one and only element in B, denoted f(x).

Equality of function: Two functions

f : A ⟼ B and g : A ⟼ B
are equal if f(x) = g(x) for all x ∈ A.

domain A of f : A ⟼ B
range the set { f(x) : x ∈ A }
one-to-one function if f(x) = f(y) implies x = y
onto functions the range { f(x) : x ∈ A } = B
functions need not be one-to-one
with x ≠ y, it is allowed that f(x) = f(y)
functions need not be onto
in general, the range { f(x) : x ∈ A } ⊆ B

Definition (composite): Let A, B, and C be sets and

f : A ⟼ B and g : B ⟼ C
be functions. Then a function composite of g and f is denoted
g∘f : A ⟼ C ⋯ (g∘f)(x) = g(f(x)) for all x ∈ A

For example, let A = B = C = ℝ,

f(x) = sin x and g(x) = x2+3
then
g∘f : ℝ ⟼ ℝ ⋯ (g∘f)(x) = (g(f(x))) = (sin x)2+3
f∘g : ℝ ⟼ ℝ ⋯ (f∘g)(x) = (f(g(x))) = sin(x2+3)

Proposition (composite associativity): Functional composition is associative. Let f, g, and h be functions, then

h∘(g∘f) = (h∘g)∘f
Proof: We prove that both sides are equal for every input x.
(h∘(g∘f))(x) = h((g∘f)(x)) ⋯ composite of h and g∘f
= h(g(f(x))) ⋯ composite of g and f
= (h∘g)(f(x)) ⋯ h∘g that takes f(x) as input
= ((h∘g)∘f)(x) ⋯ composite of h∘g and f

Notation (example 3: #x1D4D5;(S, F), a set of functions) : Let S be any nonempty set and F be any field, and let 𝓕(S, F) denote the set of all functions from S to F.

𝓕(S, F) = { f : S ⟼ F }
Two functions f and g in 𝓕(S, F) are called equal if
f(s) = g(s) for each s ∈ S

Proposition (functions as a vector space): The set 𝓕(S, F) is a vector space with the operations of addition and scalar multiplication, for f, g ∈ 𝓕(S, F) and c ∈ F by, for each s ∈ S

(f+g)(s) = f(s)+g(s) ⋯ (VA)
(cf)(s) = c(f(s)) ⋯ (SM)
Proof: We show that functions f, g, h ∈ 𝓕(S, F) satisfy vector space axioms. Let a, b ∈ F.
vector addition scalar multiplication
commutativity f+g = g+f
associativity (f+g)+h = f+(g+h) (ab)f = a(bf)
existence of identity ∃0 ∈ 𝓕, f+0 = f 1f = f
existence of inverse ∃(f−1) ∈ 𝓕, f+(f−1) = 0
distributivity a(f+g) = af+ag (a+b)f = af+bf

Commutativity of vector addition:

(f+g)(s) = f(s)+g(s) by definition (VA)
= g(s)+f(s) since f(s), g(s) ∈ F
= (g+f)(s) by definition (VA)
Associativity of vector addition:
((f+g)+h)(s) = (f+g)(s)+h(s) by definition (VA)
= (f(s)+g(s))+h(s) by definition (VA)
= f(s)+(g(s)+h(s)) since f(s), g(s), h(s) ∈ F
= f(s)+(g+h)(s) by definition (VA)
= (f+(g+h))(s) by definition (VA)
Associativity of scalar multiplication:
(ab)f(s) = a(bf(s)) since a, b, f(s) ∈ F
Existence of additive identity:

Since F is a field, F contains 0, the zero vector. Hence 𝓕 should contain a function that maps an element in S to 0. Let 0(s) denote the function such that 0(s) = 0.

(f+0)(s) = f(s)+0(s) by definition (VA)
= f(s)+0 since 0(s) = 0
= f(s) by definition of 0 ∈ F
Existence of scalar multiplicative identity:
(1f)(s) = 1f(s) by definition (SM)
= f(s) by definition of 1 ∈ F
Existence of inverse:

Since F is a field, F contains both f(s) and an inverse of f(s), denoted −f(s). Hence 𝓕 should contain a function that maps an element s ∈ S to −f(s). Let f−1(s) denote the function such that f−1(s) = −f(s).

(f+(f−1))(s) = f(s)+f−1(s) by definition (VA)
= f(s)+(−f(s)) since f−1(s) = −f(s)
= 0 since −f(s) is an additive inverse of f(s)
Distributivity over vector addition:
(a(f+g))(s) = a(f+g)(s) by definition (SM)
= a(f(s)+g(s)) by definition (VA)
= af(s)+ag(s) since a, f(s), g(s) ∈ F
Distributivity over scalar multiplication:
((a+b)f)(s) = (a+b)f(s) by definition (SM), note that (a+b) ∈ F
= af(s)+bf(s) since a, b, f(s) ∈ F

Polynomials

Polynomial [多項式]: A polynomial with coefficients [係数] from a field F is an expression [式] of the form

f(x) = an⁢xn + an-1⁢xn-1 + ⋯ + a1⁢x + a0
The degree [次数] of a polynomial is defined to the largest exponent of x with a non-zero coefficient.

Zero polynomial:

  • is a polynomial with an, an−1, …, a1, a0 = 0
  • that is f(x) = 0
  • its degree is defined to be −1

Polynomials with degree zero: are written in the form f(x) = c for some nonzero scalar c.

Equality of polynomials: Two polynomials,

f(x) = an⁢xn + an-1⁢xn-1 + ⋯ + a1⁢x + a0
g(x) = bm⁢xm + bm-1⁢xm-1 + ⋯ + b1⁢x + b0
are called equal if m = n and ai = bi for i = 0, 1, …, n.

Interpretation (a polynomial as a function): With a field F containing infinitely many scalars, we usually regard

a polynomial with coefficients from F as a function: F ⟼ F

Notation (P(F): a set of polynomials): A set of all polynomials with coefficients from a field F is denoted P(F).

Proposition (P(F) is a vector space): P(F) is a vector space with the two operations — suppose m ≤ n, and bm+1 = bm+2 = ⋯ = bn = 0, then

f(x)+g(x) = (an+bn)⁢xn + (an-1+bn-1)⁢xn-1 + ⋯ + (a1+b1)⁢x + (a0+b0)
c⁢f(x) = c⁢an⁢xn + c⁢an-1⁢xn-1 + ⋯ + c⁢a1⁢x + c⁢a0
Proof: similar to the proof of Proposition (functions as a vector space).

Notation (Pn(F): a set of all polynomials with degree ≤ n): Let n ≥ 0, and let Pn(F) consist of all polynomials in P(F) having degree ≤ n.

Proposition (Pn(F) is a subspace): Pn(F) is a subspace of P(F).

Proof: Let p1, p2 ∈ Pn(F), and let c ∈ F,

zero ∈ Pn(F) ∵ the zero polynomial has degree -1 < n
p1+p2 ∈ Pn(F) ∵ p1+p2 is another polynomial of degree ≤ n
cp1 ∈ Pn(F) ∵ cp1 is another polynomial of degree ≤ n
Hence by theorem (subspace conditions), Pn(F) is a subspace of P(F). ▮

Linear Combinations and Systems of Linear Equations

Systems of linear equations [連立一次方程式 | 線型方程式系] : A system of linear equations consists of two or more linear equations made up of two or more variables such that all equations in the system are considered simultaneously.

To find the unique solution [解] to a system of linear equations, we must find a numerical value for each variable in the system that will satisfy all equations in the system at the same time. In order for a linear system to have a unique solution, there must be at least as many equations as there are variables. Even so, this does not guarantee a unique solution. ( Systems of Linear Equations, LibreTexts Mathematics)

Example (1.4.2: linear combination of polynomials) : Consider linear combinations of two polynomials

x3−2x2−5x−3
3x3−5x2−4x−9
Case 1: 2x3−2x2+12x−6 is a linear combination of the polynomials in P3(ℝ)

2x3−2x2+12x−6 = a(x3−2x2−5x−3) + b(3x3−5x2−4x−9)
= (a+3b)x3 + (−2a−5b)x2 + (−5a−4b)x + (−3a−9b)
Thus we are led to the following system of linear equations, adding appropriate multiples of the first equation to the others in order to eliminate a, and adding the appropriate multiples of the second equation to the others yields a and b:
a+3b =2 ⟶ a+3b =2 ⟶ a =−4
−2a−5b =−2 b =2 b =2
−5a−4b =12 11b =22 0 =0
−3a−9b =−6 0b =0 0 =0
Hence 2x3−2x2+12x−6 is a linear combinations of the two polynomials:
2x3−2x2+12x−6 = −4(x3−2x2−5x−3) + 2(3x3−5x2−4x−9)

Case 2: 3x3−2x2+7x+8 is not a linear combination of the polynomials

3x3−2x2+7x+8 = a(x3−2x2−5x−3) + b(3x3−5x2−4x−9)
Using the preceding technique, we obtain a system of linear equations, and eliminating a as before yields
a+3b =3 ⟶ a+3b =3
−2a−5b =−2 b =4
−5a−4b =7 11b =22
−3a−9b =8 0 =17
But the presence of the inconsistent equation 0 = 17 indicates that the system of linear equations has no solution. Hence 3x3−2x2+7x+8 is not a linear combination of the two polynomials.
