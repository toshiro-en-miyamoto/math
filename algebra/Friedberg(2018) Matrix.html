Matrices in Linear Algebra [線形代数における行列]

Tuples

Definition (n-tuple): Let a1, a2, …, an be elements of a field F. We call an object of the following form an n-tuple with elements from F:

(a1, a2, …, an)
where the following statement holds: for any i, where 1 ≤ i ≤ n
(a1, a2, …, an) = (b1, b2, …, bn) ⇔ ai = bi

Notation (Fn : a set of n-tuples): A set of all n-tuples with entries from a filed F is denoted Fn, e.g.,

let u = (u1, u2, …, un), then u ∈ Fn

Proposition (example 1.2.1: Fn as a vector space): Fn is a vector space over F with the operations of vector addition and scalar multiplication; for u, v ∈ Fn and a ∈ F,

vector addition u+v = (u1+v1, u2+v2, …, un+vn) ⋯ (TA)
scalar multiplication au = (au1, au2, …, aun) ⋯ (TM)
Proof: We show that n-tuples u, v and w satisfy vector space axioms. Let w ∈ Fn and b ∈ F.
vector addition scalar multiplication
closed u+v ∈ Fn au ∈ Fn
commutativity u+v = v+u
associativity (u+v)+w = u+(v+w) (ab)u = a(bu)
existence of identity ∃0 ∈ Fn, u+0 = u 1u = u
existence of inverse ∃(−u) ∈ Fn, u+(−u) = 0
distributivity a(u+v) = au+av (a+b)u = au+bu

Closed vector addition:

u+v = (u1+v1, u2+v2, …, un+vn) by definition (TA)
∈ Fn ∵ u1+v1, u2+v2, …, un+vn ∈ F
Closed scalar multiplication:
au = (au1, au2, …, aun) by definition (TM)
∈ Fn ∵ au1, au2, …, aun ∈ F
Commutativity of vector addition:
u+v = (u1+v1, u2+v2, …, un+vn) by definition (TA)
= (v1+u1, v2+u2, …, vn+un) ∵ ui, vi ∈ F
= v+u by definition (TA)
Associativity of vector addition:
(u+v)+w
= (u1+v1, u2+v2, …, un+vn) + w by definition (TA)
= ((u1+v1)+w1, (u2+v2)+w2, …, (un+vn)+wn) by definition (TA)
= (u1+(v1+w1), u2+(v2+w2), …, un+(vn+wn)) ∵ ui, vi, wi ∈ F
= u+(v+w) by definition (TA)
Associativity of scalar multiplication:
(ab)u = ((ab)u1, (ab)u2, …, (ab)un) by definition (TM)
= (a(bu1), a(bu2), …, a(bun)) ∵ ui, a, b ∈ F
= a(bu1, bu2, …, bun) by definition (TM)
= a(bu) by definition (TM)
Existence of additive identity:
Let 0 = (0, 0, …, 0) ∈ Fn, then
u+0 = (u1+0, u2+0, …, un+0) by definition (TA)
= (u1, u2, …, un) ∵ 0 is additive identity in F
= u by definition
Existence of multiplicative identity:
1u = (1u1, 1u2, …, 1un) by definition (TM)
= (u1, u2, …, un) ∵ 1 is multiplicative identity in F
= u by definition
Existence of additive inverse:
Let −u = (−u1, −u2, …, −un), then
u+(−u)
= (u1+(−u1), u2+(−u2), …, un+(−un)) by definition (TA)
= (0, 0, …, 0) ∵ −ui is an additive identity of ui in F
= 0 by definition
Distributivity over tuple addition:
a(u+v) = a(u1+v1, u2+v2, …, un+vn) by definition (TA)
= (a(u1+v1), a(u2+v2), …, a(un+vn)) by definition (TM)
= (au1+av1, au2+av2, …, aun+avn) ∵ ui, vi, a ∈ F
= (au1, au2, …, aun) + (av1, av2, …, avn) by definition (TA)
= au+av by definition (TM)
Distributivity over scalar addition:
(a+b)u = ((a+b)u1, (a+b)u2, …, (a+b)un) by definition (TM)
= (au1+bu1, au2+bu2, …, aun+bun) ∵ ui, a, b ∈ F
= (au1, au2, …, aun) + (bu1, bu2, …, bun) by definition (TA)
= au+bu by definition (TM) ▮

Notation (column vector): Vectors in Fn may be written as column vectors:

( a1 a2 ⋮ an )
rather than as row vectors ( a1, a2, …, an )

Proposition (exercise 1.3.24: Fn as a direct sum) : A set of n-tuples Fn is the direct sum of subspaces:

W1 = { (a1, a2, …, an) ∈ Fn : an = 0 }
W2 = { (a1, a2, …, an) ∈ Fn : a1 = a2 = ⋯ = an−1 = 0 }
Proof: We show that (1) W1 and W2 are subspaces of Fn, and (2) W1 ∩ W2 = {0} and W1+W2 = Fn.

(1a) W1 is a subspace:

By definition of W1, it contains the zero tuple (0, 0, …, 0) where a1 = a2 = ⋯ = an−1 = 0.

Let r, s ∈ W1 such that

r = (a1, a2, …, an-1, 0) s = (b1, b2, …, bn-1, 0)
where ai and bi are scalars. Then the vector addition(TA) and the scalar multiplication (TM) defined in Proposition (Fn as a vector space) allow us to have
r+s = (a1+b1, a2+b2, …, an-1+bn-1, 0)
cr = (ca1, ca2, …, can-1, 0)
for any scalar c. It is immediate that r+s and cr are tuples contained in W1. Hence W1 is closed under vector addition and scalar multiplication.

(1b) W2 is a subspace:

By definition of W2, it contains the zero tuple (0, 0, …, 0) where an = 0.

Let t, u ∈ W2 such that

t = (0, 0, …, 0, an) u = (0, 0, …, 0, bn)
where an and bn are scalars. Similarly with W1,
t+u = (0, 0, …, 0, an+bn)
ct = (0, 0, …, 0, can)
for any scalar c. It is immediate that t+u and ct are tuples contained in W2. Hence W2 is closed under vector addition and scalar multiplication.

(2) W1 ∩ W2 = {0} and W1+W2 = Fn:

Suppose that there exists a non-zero tuple v = (a1, a2, …, an) such that v ∈ (W1 ∩ W2). Then by definitions of W1 and W2, a1 = a2 = ⋯ = an = 0, that is, v is the zero tuple, a contradiction. Therefore, (W1 ∩ W2) doesn't contain non-zero tuples. But since W1 and W2 are subspaces, 0 ∈ (W1 ∩ W2). Hence, (W1 ∩ W2) = {0}.

By definition of W1 and W2, for some scalars ai where 1 ≤ i ≤ n,

Fn = { (a1, a2, …, an-1, an) }
= { (a1, a2, …, an-1, 0) + (0, 0, …, 0, an) }
= W1+W2 ▮

Matrices

Definition (matrix): An m×n matrix with entries from a field F is

( a11 a12 ⋯ a1n a21 a22 ⋯ a2n ⋮ ⋮ ⋮ am1 am2 ⋯ amn )
where each entry aij (1 ≤ i ≤ m, 1 ≤ j ≤ n) is an element of F.
  • ith row: ai1, ai2, …, ain
  • jth column: a1j, a2j, …, amj
  • zero matrix: denoted 0, all of its entries are 0
  • diagonal entries: aij with i = j
  • square matrix [正方行列]: n×n matrices for any n ≥ 0

We denote

  • matrices by capital letters, e.g. A, and
  • the entry of a matrix A that lies in row i and column j by Aij.

Notation (Mm×n(F): a set of matrices): The set of all m×n matrices with each entries from a field F is denoted Mm×n(F).

Proposition (example 1.2.2: Mm×n(F) as vector space) : Mm×n(F) is a vector space over F with the following operations of matrix addition and scalar multiplication; for A, B ∈ Mm×n(F) and c ∈ F, and 1 ≤ i ≤ m and 1 ≤ j ≤ n,

matrix addition (A+B)ij = Aij+Bij ⋯ (MA)
scalar multiplication (cA)ij = cAij ⋯ (MM)
Proof: very similar to the proof of Proposition (Fn as a vector space).

Definition (transpose [転置行列]): The transpose At of an m×n matrix A is the n×m matrix obtained from A by interchanging the rows with the columns; that is,

(At)ij = Aji
For example,
( 1 −2 3 0 5 −1 ) t = ( 1 0 −2 5 3 −1 )

Proposition (diagonal entries of transposes): Diagonal entries of a matrix A are equal to diagonal entries of transposed matrix At.

Proof: From the definition of transpose,

(At)ii = Aii for 1 ≤ i ≤ min(n,m) ▮

Proposition (exercise 1.3.3: distributivity of transposes): Let A, B ∈ Mm×n(F) and a, b ∈ F, then

(a⁢A + b⁢B)t = a⁢At + b⁢Bt
Proof:
((aA + bB)t)ij = (aA + bB)ji by definition of transpose
= (aA)ji + (bB)ji by definition (MA)
= ((aA)t)ij + ((bB)t)ij by definition of transpose
= a(At)ij + b(Bt)ij by definition (MM) ▮

Definition (symmetric matrix [対称行列]): A symmetric matrix is a square matrix such that

At = A that is (At)ij = Aij
For example,
( 1 2 2 3 ) t = ( 1 2 2 3 )

Proposition (exercise 1.3.5: adding transpose gets symmetric): A+At is symmetric for any square matrix A.

Proof:

((A+At)t)ij = (A+At)ji by definition of transpose
= Aji+(At)ji by definition (MA)
= Aji+Aij by definition of transpose
= Aij+Aji ∵ Aji, Aij ∈ F
= Aij+(At)ij by definition of transpose
= (A+At)ij by definition (MA)
Thus the transpose of A+At is equal to A+At, it follows that A+At is symmetric. ▮

Proposition (set of symmetric matrices as subspace): The set W of all symmetric matrices in Mn×n(F) is a subspace of Mn×n(F).

Proof: We show the subspace conditions hold for W. Let A, B ∈ W and c ∈ F,

0 ∈ W:
0 = 0t ∵ 0ji = 0ij
⁣
A+B ∈ W:
A+B = At+Bt by definition of the symmetric matrix
= (A+B)t by Proposition (distributivity of transposes)
⁣
cA ∈ W:
cA = cAt by definition of the symmetric matrix
= (cA)t by definition (MM) ▮

Upper triangular [上三角行列]: An matrix A is called upper triangular if Aij = 0 whenever i > j.

A = ( 1 2 3 4 0 5 6 7 0 0 8 9 )

Definition (diagonal matrix [対角行列]): A square matrix A is called a diagonal matrix if Aij = 0 whenever i ≠ j.

A = ( 3 0 0 0 −2 0 0 0 8 )

Proposition (exercise 1.3.7: diagonals are symmetric): Diagonal matrices are symmetric matrices.

Proof: Let A be a diagonal matrix, then

  • if i ≠ j, by the definition of diagonal matrix, (At)ij = Aji = 0 = Aij
  • if i = j, by Proposition (diagonal entries of transposes), (At)ij = Aij
Thus, since (At)ij = Aij for any i and j, diagonal matrices are symmetric. ▮

Proposition (example 1.3.3: diagonal matrices as subspace): The set of diagonal matrices W is a subspace of Mn×n(F).

Proof: Vectors in W meet the subspace conditions; let A, B ∈ W and c ∈ F,

0 ∈ W ∵ whenever i ≠ j, (0)ij = 0; thus the zero matrix is diagonal
A+B ∈ W ∵ whenever i ≠ j, (A+B)ij = Aij + Bij = 0 + 0 = 0; thus A+B is diagonal
cA ∈ W ∵ whenever i ≠ j, (cA)ij = cAij = c0 = 0; thus cA is diagonal ▮

Notation (tr(M): the trace of a square matrix): The trace of an n×n matrix M, denoted tr(M), is the sum of the diagonal entries of M; that is,

tr(M) = M11 + M22 + ⋯ + Mnn

Proposition (exercise 1.3.6: distributivity of trace): For any A, B ∈ Mn×n(F) and any a, b ∈ F,

tr(aA+bB) = a tr(A) + b tr(B)
Proof:
(aA+bB)ii = (aA)ii+(bB)ii by definition (MA)
= aAii+bBii by definition (MM)
It follows that
tr(aA+bB)
= (aA+bB)11 + (aA+bB)22 + ⋯ + (aA+bB)nn by definition of trace
= aA11+bB11 + aA22+bB22 + ⋯ + aAnn+bBnn ∵ (aA+bB)ii = aAii+bBii (see above)
= a(A11+A22+⋯+Ann) + b(B11+B22+⋯+Bnn) ∵ Aii, Bii, a, b ∈ F
= a tr(A) + b tr(B) by definition of trace ▮

Proposition (example 1.3.4: zero-trace as subspace): Let W the set of n×n matrices having trace equal to zero, then W is a subspace of Mn×n(F).

Proof: Vectors in W meet the subspace conditions; let A, B ∈ W and c ∈ F,

0 ∈ W:
tr(0) = 0+0+⋯+0 = 0 by definition of the zero matrix
⁣
A+B ∈ W:
tr(A+B) = tr(A) + tr(B) = 0+0 = 0 by Proposition (distributivity of trace)
⁣
cA ∈ W
tr(cA) = c tr(A) = c0 = 0 by Proposition (distributivity of trace)

Skew-symmetric matrix [交代行列]: A square matrix M is called skew-symmetric if

Mt = −M
For example,
M = ( 0 1 −2 −1 0 3 2 −3 0 ) Mt = ( 0 −1 2 1 0 −3 −2 3 0 ) −M = ( 0 −1 2 1 0 −3 −2 3 0 )
