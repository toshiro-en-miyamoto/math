Matrix in Linear Algebra [線形代数における行列]

Tuples

Notation (n-tuple): Let a1, a2, …, an be elements of a field F. We call an object of the following form an n-tuple with elements from F:

(a1, a2, …, an)
where the following statement holds: for any i, where 1 ≤ i ≤ n
(a1, a2, …, an) = (b1, b2, …, bn) ⇔ ai = bi

Notation (Fn : a set of n-tuples): A set of all n-tuples with entries from a filed F is denoted Fn, e.g.,

let u = (u1, u2, …, un), then u ∈ Fn

Proposition (example 1.2.1: Fn is a vector space): Fn is a vector space over F with the operations of vector addition and scalar multiplication; for u, v ∈ Fn and a ∈ F,

u+v = (u1+v1, u2+v2, …, un+vn) ⋯ (TA)
au = (au1, au2, …, aun) ⋯ (TM)
Proof: We show that n-tuples u, v and w satisfy vector space axioms. Let w ∈ Fn and b ∈ F.
vector addition scalar multiplication
closed u+v ∈ Fn au ∈ Fn
commutativity u+v = v+u
associativity (u+v)+w = u+(v+w) (ab)u = a(bu)
existence of identity ∃0 ∈ Fn, u+0 = u 1u = u
existence of inverse ∃(−u) ∈ Fn, u+(−u) = 0
distributivity a(u+v) = au+av (a+b)u = au+bu

Closed vector addition:

u+v = (u1+v1, u2+v2, …, un+vn) by definition (TA)
∈ Fn ∵ u1+v1, u2+v2, …, un+vn ∈ F
Closed scalar multiplication:
au = (au1, au2, …, aun) by definition (TM)
∈ Fn ∵ au1, au2, …, aun ∈ F
Commutativity of vector addition:
u+v = (u1+v1, u2+v2, …, un+vn) by definition (TA)
= (v1+u1, v2+u2, …, vn+un) ∵ ui, vi ∈ F
= v+u by definition (TA)
Associativity of vector addition:
(u+v)+w
= (u1+v1, u2+v2, …, un+vn) + w by definition (TA)
= ((u1+v1)+w1, (u2+v2)+w2, …, (un+vn)+wn) by definition (TA)
= (u1+(v1+w1), u2+(v2+w2), …, un+(vn+wn)) ∵ ui, vi, wi ∈ F
= u+(v+w) by definition (TA)
Associativity of scalar multiplication:
(ab)u = ((ab)u1, (ab)u2, …, (ab)un) by definition (TM)
= (a(bu1), a(bu2), …, a(bun)) ∵ ui, a, b ∈ F
= a(bu1, bu2, …, bun) by definition (TM)
= a(bu) by definition (TM)
Existence of additive identity:
Let 0 = (0, 0, …, 0) ∈ Fn, then
u+0 = (u1+0, u2+0, …, un+0) by definition (TA)
= (u1, u2, …, un) ∵ 0 is additive identity in F
= u by definition
Existence of multiplicative identity:
1u = (1u1, 1u2, …, 1un) by definition (TM)
= (u1, u2, …, un) ∵ 1 is multiplicative identity in F
= u by definition
Existence of additive inverse:
Let −u = (−u1, −u2, …, −un), then
u+(−u) = (u1+(−u1), u2+(−u2), …, un+(−un)) by definition (TA)
= (0, 0, …, 0) ∵ −ui is an additive identity of ui in F
= 0 by definition
Distributivity over tuple addition:
a(u+v) = a(u1+v1, u2+v2, …, un+vn) by definition (TA)
= (a(u1+v1), a(u2+v2), …, a(un+vn)) by definition (TM)
= (au1+av1, au2+av2, …, aun+avn) ∵ ui, vi, a ∈ F
= (au1, au2, …, aun) + (av1, av2, …, avn) by definition (TA)
= au+av by definition (TM)
Distributivity over scalar addition:
(a+b)u = ((a+b)u1, (a+b)u2, …, (a+b)un) by definition (TM)
= (au1+bu1, au2+bu2, …, aun+bun) ∵ ui, a, b ∈ F
= (au1, au2, …, aun) + (bu1, bu2, …, bun) by definition (TA)
= au+bu by definition (TM)

Notation (column vector): Vectors in Fn may be written as column vectors:

( a1 a2 ⋮ an )
rather than as row vectors ( a1, a2, …, an )

Matrices

Notation (matrix): An m×n matrix with entries from a field F is

( a11 a12 ⋯ a1n a21 a22 ⋯ a2n ⋮ ⋮ ⋮ am1 am2 ⋯ amn )
where each entry aij (1 ≤ i ≤ m, 1 ≤ j ≤ n) is an element of F.
  • ith row: ai1, ai2, …, ain
  • jth column: a1j, a2j, …, amj
  • zero matrix: denoted 0, all of its entries are 0
  • diagonal entries: aij with i = j
  • square matrix [正方行列]: n×n matrices for any n ≥ 0

We denote

  • matrices by capital letters, e.g. A, and
  • the entry of a matrix A that lies in row i and column j by Aij.

Notation (Mm×n(F): a set of matrices): The set of all m×n matrices with each entries from a field F is denoted Mm×n(F).

Proposition (example 1.2.2: Mm×n(F) is a vector space) : Mm×n(F) is a vector space over F with the following operations of matrix addition and scalar multiplication; for A, B ∈ Mm×n(F) and c ∈ F, and 1 ≤ i ≤ m and 1 ≤ j ≤ n,

(A+B)ij = Aij+Bij
(cA)ij = cAij
Proof: very similar to the proof of Example 1.2.1.

Notation (transpose [転置行列]): The transpose At of an m×n matrix A is the n×m matrix obtained from A by interchanging the rows with the columns; that is,

(At)ij = Aji
For example,
( 1 −2 3 0 5 −1 ) t = ( 1 0 −2 5 3 −1 )

Lemma (distributivity of transposes): Let A, B ∈ Mm×n(F) and a, b ∈ F, then

(a⁢A + b⁢B)t = a⁢At + b⁢Bt
Proof:
((aA + bB)t)ij = (aA + bB)ji ∵ (At)ij = Aji
= (aA)ji + (bB)ji ∵ (A+B)ij = Aij + Bij
= ((aA)t)ij + ((bB)t)ij ∵ (At)ij = Aji
= a(At)ij + b(Bt)ij ∵ (cA)ij = cAij ▮

Symmetric matrix [対称行列]: A symmetric matrix is a matrix such that

At = A
For example,
( 1 2 2 3 ) t = ( 1 2 2 3 )
Proposition (set of symmetric matrices as subspace): The set W of all symmetric matrices in Mn×n(F) is a subspace of Mn×n(F), because the subspace conditions hold for W. Let A, B ∈ W and a ∈ F,
0 ∈ W ∵ 0 = 0t ∈ W
A+B ∈ W ∵ A+B = At+Bt = (A+B)t ∈ W
aA ∈ W ∵ aA = aAt = (aA)t ∈ W

Upper triangular [上三角行列]: An m×n matrix A is called upper triangular if Aij = 0 whenever i > j.

A = ( 1 2 3 4 0 5 6 7 0 0 8 9 )

Diagonal matrix [対角行列]: A square matrix A is called a diagonal matrix if Aij = 0 whenever i ≠ j.
A = ( 3 0 0 0 −2 0 0 0 8 )

The set of diagonal matrices W is a subspace of Mn×n(F), because the subspace conditions hold for W. Let A, B be diagonal matrices and let c ∈ F,

0 ∈ W ∵ the zero matrix is diagonal
A+B ∈ W ∵ whenever i ≠ j, (A+B)ij = Aij + Bij = 0 + 0 = 0
cA ∈ W ∵ whenever i ≠ j, (cA)ij = cAij = c0 = 0

Skew-symmetric matrix [交代行列]: A square matrix M is called skew-symmetric if

Mt = −M
For example,
M = ( 0 1 −2 −1 0 3 2 −3 0 ) Mt = ( 0 −1 2 1 0 −3 −2 3 0 ) −M = ( 0 −1 2 1 0 −3 −2 3 0 )
