4. Counting Principles

4.1. A Fundamental Counting Principle

Fundamental Counting Principle: Consider an activity consisting of k independent subtasks. Assume that

  • task₁ can be performed in m₁ ways,
  • task₂ can be performed in m₂ ways,
  • ⋮
  • taskk can be performed in mk ways.
Then this activity can be completed in m₁ × m₂ × ⋯ × mk different number of ways.

Example 4.2: Find the number of all possible six-digit numbers that can be formed by using 2, 6, 8 and 9.

Solution:

  • The unit place can be filled by any of the four given digits,
  • the tenth place can be filled by any of the four given digits,
  • ⋮
  • and so on upto the hundred-thousandth place.
Hence, the required answer is 46.

Example 4.7: Show that the number of ways of placing n distinct objects into m boxes is mn.

Solution:

  • The first object go into any one of the m boxes. Clearly, there are m ways of doing this.
  • Similarly, the second object can also be placed in any of the m boxes and so on.
Hence, the required answer is mn.

Note: If we think of the place as boxes and the digits as objects, the answer is 64. What's wrong?

Proposition 4.1: Let A and B be finite sets with |A| = n and |B| = m. Then the number of all functions f : A → B is mn.

Proof:

Let A = {x₁, x₂, ⋯, xn}. Then

  • f(x₁) can be defined in m possible ways
  • f(x₂) can be defined in m possible ways
  • ⋮
  • f(xn) can be defined in m possible ways

Thus, the number of all possible functions is mn.

Definition (function): A function, f : A → B, is a rule that associates each element x in A with an element in B, denoted f(x). It demands that

  • x₁ = x₂ implies f(x₁) = f(x₂)
  • for all x ∈ A there exists f(x) such that f(x) ∈ B
Note that
  • even if f(x₁) = f(x₂), it is not necessarily that x₁ = x₂
  • there exists y ∈ B such that y ∉ { f(x) : x ∈ A }

Functional solution to example 4.7: Consider one specific distribution of the n objects into m distinct boxes.

  • Think of a relation from objects to boxes. Because each object is placed in one of boxes, a function f : objects → boxes can be defined.
  • Think of a relation from boxes to objects. Some boxes can be empty, meaning that some boxes are not related to objects. Hence, the relation cannot be a function.
Thus, the number of all possible f : objects → boxes = mn is the required answer.

Functional solution to example 4.2: Consider a six-digit number that can be formed by using 2, 6, 8 and 9.

  • Think of a relation from digits to places. Some digits may not appear in some number (e.g., 226688), meaning that some digits are not related to places. Hence, the relation cannot be a function.
  • Think of a relation from places to digits. Because each place is filled with one of digits, a function f : places → digits can be defined.
Thus, the number of all possible f : places → digits = 46 is the required answer.

4.1.1. Arrangements

In how many ways can three objects be arranged in some order? Let us call these objects as a, b, c. We can arrange them as abc, acb, bac, bca, cab, cba. These are all the possible ways and hence three objects can be arranged in six different ways.

Proposition 4.2: r objects can be arranged in r! ways.

Proof:

Imagine an array of r boxes. Consider the filling of each of the r boxes as a subtask.

  • We can place any of the r objects in the first box. Thus, the number of ways of completing this first task is r.
  • Now we are left with r − 1 objects. So the second box can be filled in r − 1 ways.

Thus, by using the counting principle of the previous section, the number of different arrangements of r things is r × (r − 1) × ⋯ × 1 = r!.

We consider problems involving special boxes into which only certain special objects can be placed. Consider a row of r boxes. Assume that

  • only m₁ of the objects can be placed in the first box, after which
  • only m₂ of the remaining objects can be placed in the second box, and lastly
  • only mr of the remaining objects can be placed in the r th box.

The number of arrangements is m₁ × m₂ × ⋯ × mr.

4.2. Permutations and Combinations

Definition (permutation): Let there be n objects. The number of ways of choosing and arranging r objects out of the n objects is called the permutation of r objects out of n, denoted nPr.

Proposition 4.3:

nPr = n(n − 1)(n − 2)⋯(n − r + 1) = n! ÷ (n − r)!
Proof:

Consider an array of r boxes. We can place

  • any of the n − 0 objects in the first box
  • any of the remaining n − 1 objects in the second box
  • ⋮
  • any of the remaining n − r + 1 objects in the r th box

Hence, the number of ways this can be achieved is n(n − 1)(n − 2)⋯(n − r + 1).

Example 4.18: Consider five-letter words that can be formed using nine letters A, B, ⋯, I without repetition. How many of them contain A and I?

Solution:

  • First, we shall allot two places out of five to A and I. This can be done in 5P2 ways.
  • Next, we shall fill the remaining three places with seven letters B, C, ⋯ H without repetition. This can be done in 7P3.

Hence, the required answer is 5P2 × 7P3.

Example 4.19:

  • In how many ways can five letters be chosen from nine letters A, B, ⋯, I?
  • How many of them contain A and I?
In Example 4.18, the order in which the letters appear is important. But in this example, letters are just selected without any consideration for the order. We need the idea of combination.

Definition (combination): Let there be n objects. The number of ways of choosing r objects out of n objects is called the combination of r objects out of n objects, denoted nCr.

Proposition 4.4:

nCr = nPr ÷ r!
Proof:

Choosing and arranging r objects out of n objects consists of two subtasks:

  • choosing r objects out of n objects — in nCr ways
  • arranging r objects — in r! ways

Thus, nPr = nCr × r!.

Example 4.19:

In how many ways can five letters be chosen from nine letters A, B, ⋯, I? — It is the number of ways of choosing five out of nine = 9C5.

How many of them contain A and I? — If our choice must include both A and I, then we need to choose only three out of seven = 7C3.

4.2.2. Permutation of Identical Objects

Consider nine-digit numbers that are formed by two 4's, three 6's and four 8's, such as 446668888. How many distinct numbers can we have?

Method 1a:

  • Choose two places for two 4's out of the nine places — in 9C2.
  • Choose three places for three 6's out of the remaining seven places — in 7C3.
  • Choose four places for four 8's out of the remaining four places — in 4C4.
  • Hence, the answer is ( 9 × 8 2 × 1 ) ⁢ ( 7 × 6 × 5 3 × 2 × 1 ) ⁢ ( 4 × 3 × 2 × 1 4 × 3 × 2 × 1 ) = 1,260 distinct numbers.

Method 1b:

  • Choose four places for two 8's out of the nine places — in 9C4.
  • Choose three places for three 6's out of the remaining five places — in 5C3.
  • Choose two places for three 2's out of the remaining two places — in 2C2.
  • Hence, the answer is ( 9 × 8 × 7 × 6 4 × 3 × 2 × 1 ) ⁢ ( 5 × 4 × 3 3 × 2 × 1 ) ⁢ ( 2 × 1 2 × 1 ) = 1,260 distinct numbers.

Methods 2:

  • Think of 4's, 6's and 8's as nine distinct objects: 4₁, 4₂, 6₁, 6₂, 6₃, 8₁, 8₂, 8₃, 8₄. Then the objects can be arranged in 9P9 ways (= 9!).
  • We must no distinguish two 4's. But among the arrangements, we counted the identical nine-digit numbers 2P2 times (= 2!).
  • We must not distinguish three 6's. But among the arrangements, we counted the identical nine-digit numbers 3P3 times (= 3!).
  • We must not distinguish four 8's. But among the arrangements, we counted the identical nine-digit numbers 4P4 times (= 4!).
  • Hence, the answer is 9! 2! × 3! × 4! = 1,260 distinct numbers.

As you can see the three methods yield the correct answer. The last method is based on the idea of permutation of identical objects.

Proposition 4.6: In a collection there are m₁ identical articles of type 1, m₂ articles of type 2 and so on upto mk articles of type k. The number of permutation of this collection is

( m1 + m2 + ⋯ + mk ) ! m1 × m2 × ⋯ × mk

Example 4.33: A hotel has six single rooms, six double rooms, and four three-bed rooms (30 beds in total). In how many ways can 30 persons be accommodated in this hotel?

Solution 1: using the method 1

  • For single rooms, six persons out of 30 persons can be chosen and arranged — in 30P6 ways.
  • For double rooms:
    • two persons out of the remaining 24 persons can be chosen for the 1st double room — in 24C2 ways.
    • two persons out of the remaining 22 persons can be chosen for the 2nd double room — in 22C2 ways.
    • 20C2 for 3rd, 18C2 for 4th, 16C2 for 5th, and 14C2 for 6th.
  • For three-bed rooms:
    • three persons out of the remaining 12 persons can be chosen for the 1st three-bed room — in 12C3 ways.
    • three persons out of the remaining 9 persons can be chosen for the 2nd three-bed room — in 9C3 ways.
    • 6C3 for the 3rd and 3C3 for the 4th.
  • Thus, the required answer is ( 30· 29· 28· 27· 26· 25 ) ⁢ ( 24· 23 2· 1 ) ⁢ ( 22· 21 2· 1 ) ⁢ ( 20· 19 2· 1 ) ⁢ ( 18· 17 2· 1 ) ⁢ ( 16· 15 2· 1 ) ⁢ ( 14· 13 2· 1 ) ⁢ ( 12· 11· 10 3· 2· 1 ) ⁢ ( 9· 8· 7 3· 2· 1 ) ⁢ ( 6· 5· 4 3· 2· 1 ) ⁢ ( 3· 2· 1 3· 2· 1 )
    • = P 6 30 · P 2 24 2! · P 2 22 2! · P 2 20 2! · P 2 18 2! · P 2 16 2! · P 2 14 2! · P 3 12 3! · P 3 9 3! · P 3 6 3! · P 3 3 3!
    • = 30! 2! · 2! · 2! · 2! · 2! · 2! · 3! · 3! · 3! · 3!

Solution 2: using the method 2

  • We shall denote
    • six single rooms by Sa, Sb, Sc, Sd, Se, Sf
    • six double rooms by Da, Db, Dc, Dd, De, Df
    • four three-bed rooms by Ta, Tb, Tc, Td
  • We shall denote beds by
    • Sa₁ | Sb₁ | Sc₁ | Sd₁ | Se₁ | Sf₁
    • Da₁, Da₂ | Db₁, Db₂ | Dc₁, Dc₂ | Dd₁, Dd₂ | De₁, De₂ | Df₁, Df₂
    • Ta₁, Ta₂, Ta₃ | Tb₁, Tb₂, Tb₃ | Tc₁, Tc₂, Tc₃ | Td₁, Td₂, Td₃
  • The 30 beds can be arranged for 30 persons in 30! ways.
  • For double rooms,
    • the beds Da₁ and Da₂ are in the room Da, so these must not be distinguished for this solution. But among the arrangement, we counted the beds 2P2 times.
    • The same goes for Db, Dc, Dd, De, Df.
  • For three-bed rooms,
    • the beds Ta₁, Ta₂, Ta₃ are in the room Ta, so these must not be distinguished for this solution. But among the arrangement, we counted the beds 3P3 times.
    • The same goes for Tb, Tc, Td.
  • Thus, the required answer is 30! 2! · 2! · 2! · 2! · 2! · 2! · 3! · 3! · 3! · 3!

4.3. Applications of Permutations and Combinations

Proposition 4.7: Let A and B be finite sets with |A| = n and |B| = m, and n ≤ m. The number of all one-to-one functions from A to B is m(m − 1)(m − 2)⋯(m − n + 1) = mPn.

Proof:

Let A = {x₁, x₂, ⋯, xn}. f(x₁) can be defined in m possible ways. Since for a one-to-one function, f(x₁) ≠ f(x₂), f(x₂) can be defined in m − 1 possible ways and so on. Thus, the number of all possible functions = m(m − 1)(m − 2)⋯(m − n + 1) = mPn.

Proposition 4.8: The number of binary sequences with p 1's and q 0's is ( p+q ) ! p! · q!

Proof: Apply Proposition 4.6.

Proposition 4.9: The number of binary sequences with p 1's and q 0's such that no two 0's are together is p+1Cq.

Proof:

First, arrange the p 1's. The 0's can be placed in the p − 1 gaps, or at the beginning or at the end. Thus, total number of places is p + 1. Then, we shall choose two places out of the p + 1 places. Hence, the required number is p+1Cq.

Example 4.35:

There is an unlimited supply of apples, oranges and bananas. You are free to take any number of any of the three kinds but the total must be exactly 10. The number of choices is 10+3−1C3−1.

Let us pretend that each type of fruit represents a box. If you choose three apples, four oranges and three bananas, then this is similar to placing three objects in the apple box, four in the orange box and three in the banana box.

Proposition 4.13: The number of ways of distributing n identical objects in k distinct non-empty boxes, with n ≥ k, is n−1Ck−1.

Proof:

Modeled as stars and bars, k − 1 bars divide stars into k groups. In contrast to Proposition 4.11, however, each group must not be empty. Therefore, the locations where bars can be placed are limited such that:

  • bars cannot be placed at the ends
  • two or more bars cannot be placed at the same location
It follows that:
  • the number of potential locations for bars is n − 1
  • k − 1 locations shall be chosen out of n − 1
Hence, the required answer is n−1Ck−1.

Example 4.38: Find the number of staircase paths from (2, 5) to (8, 9).

Proof:

Let steps R and U such that:

  • R: from (x, y) to (x + 1, y)
  • U: from (x, y) to (x, y + 1)
Now to reach (8, 9) from (2, 5), we have to move 8 − 2 = 6 units horizontally and 9 − 5 = 4 units vertically. Thus, with every staircase path, we can associate a sequence of R's and U's consisting of six R's and four U's, for example R-R-R-U-U-R-R-R-U-U. Hence, the required answer if the number of permutations of six R's and four U's, which is given by 10! 6! · 4! . (Refer to Proposition 4.6)
