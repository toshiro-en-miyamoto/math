Fields (Appendix C)

Definition (field [体]): A field is a set on which two operations + (addition) and · (multiplication) are defined so that, for each pair of elements x, y in F, there are unique elements in F, denoted x+y and x·y, and such that the following conditions hold for all elements a, b, c in F.

The field axioms [体の公理]
addition multiplication
交換則 commutativity a+b = b+a a·b = b·a
結合則 associativity (a+b)+c = a+(b+c) (a·b)·c = a·(b·c)
単位元 existence of identity 0+a = a 1·a = a
逆元 existence of inverse a+(−a) = 0 a·a-1 = 1 (if a≠0)
分配則 distributivity of · over + a·(b+c) = a·b+a·c
zero-one law 0 ≠ 1
For the zero-one law as an field axiom, refer Mayer (2006).

Theorem (cancellation laws) [簡約律]: Let F be a field. For all elements a, b, c, d ∈ F,

1. a+b = c+b ⇒ a = c
2. a·d = c·d and d≠0 ⇒ a = c
Proof:
  1. Existence of additive inverse of the field axioms guarantees that there exists an element −b ∈ F.
    (a+b)+(−b) = (c+b)+(−b) by adding (−b) to both sides
    a+{b+(−b)} = c+{b+(−b)} by associativity of addition
    a+0 = c+0 by existence of additive inverse
    a = c by existence of additive identity
  2. Existence of multiplicative inverse of the field axioms guarantees that if d≠0, there exists an element d-1 ∈ F.
    (a·d)·d-1 = (c·d)·d-1 by multiplying both sides by d-1
    a·(d·d-1) = c·(d·d-1) by associativity of multiplication
    a·1 = c·1 by existence of multiplicative inverse
    a = c by existence of multiplicative identity

Corollary (unique identities and inverses): Let F be a field and a ∈ F, then

  • Additive identity (0) is unique.
  • Additive inverse (−a) is unique.
  • Multiplicative identity (1) is unique.
  • Multiplicative inverse (a-1 if a≠0) is unique.
Proof:
  • Additive identity (0): Suppose that 0′ ∈ F is an additive identity.
    0′ + a = a since 0′ is an additive identity
    = 0 + a since 0 is an additive identity
    Hence 0′ = 0 by the additive cancellation law.
  • Other three proofs are similar.

Subtraction (−) and division (÷ or ⁄) can be defined in terms of addition and multiplication by using the additive and multiplicative inverses. Let F be a field. For all a ∈ F and all b≠0 ∈ F,

  • a−b = a+(−b)
  • a÷b = a⁄b = a·b-1

Corollary (multiplications in fields): Let F be a field. For all a, b ∈ F,

  1. a·0 = 0
  2. (−a)·b = a·(−b) = −(a·b)
  3. (−a)·(−b) = a·b
Proof:
  1. Since
    0+a·0 = a·0 by existence of additive identity
    = a·(0+0) since 0 = 0+0
    = a·0+a·0 by distributivity of · over +
    Hence a·0 = 0 by the additive cancellation law.
  2. Since
    a·b+(−a)·b = {a+(−a)}·b by distributivity of · over +
    = 0·b by existence of additive identity
    = 0 by corollary 1
    Hence (−a)·b = −(a·b) by existence of additive inverse. The proof that a·(−b) = −(a·b) is similar.
  3. By applying (2) twice,
    (−a)·(−b) = −{a·(−b)}
    = −{−(a·b)}
    = a·b

Zero ring: The zero ring or trivial ring is the unique ring (up to isomorphism) consisting of one element [Wikipedia: Zero ring]. If additive identity 0 equals multiplicative identity 1, then the ring R has only a single element 0 = 1, i.e.,

0 = 1 ⇔ R = {0}
Proof: If 0 = 1, for all x ∈ R
x = 1·x by existence of multiplicative identity
= 0·x since 0 = 1
= 0 by corollary (multiplications in fields, no.1)

Vector Spaces [ベクトル空間]

Definition (vector space): A vector space V over a field F consists of a set on which two operations (addition [加法] and scalar multiplication [スカラー倍]) are defined so that:

  • for each pair of elements x, y ∈ V, there is a unique element x+y ∈ V, and
  • for each element a ∈ F and for each element x ∈ V, there is a unique element a⁢x ∈ V,
such that the following conditions hold:

addition scalar multiplication
commutativity x+y = y+x
associativity (x+y)+z = x+(y+z) (ab)x = a(bx)
existence of identity x+0 = x 1x = x
existence of inverse x+(−x) = 0
distributivity a(x+y) = ax+ay (a+b)x=ax+bx

We call:

scalars
the elements of the field F
vectors
the elements of the vector space V
zero vector
0, the additive identity; also appeared in existence of inverse

Theorem (cancellation law for vector addition): Let V be a vector space, and x, y, z ∈ V, then

x + z = y + z ⇒ x = y
Proof: There is u ∈ V such that z + u = 0 (existence of additive inverse), then
x = x + 0 existence of additive identity
= x + (z + u) since z + u = 0
= (x + z) + u associativity of addition
= (y + z) + u since x + z = y + z
= y + (z + u) associativity of addition
= y + 0 since z + u = 0
= y existence of additive identity

Corollary (unique zero vector): The zero vector is unique.

Proof: Let V be a vector space, and x ∈ V. Suppose that 0′ ∈ V is an zero vector, too. Then,

0′ + x = x since 0′ is an zero vector
= 0 + x since 0 is an zero vector
Hence 0′ = 0 by the cancellation law.

Theorem (scalar multiplications in vector spaces): In any vector space V over a field F, the following statements are true:

  1. 0⁢x = 0 for each x ∈ V
  2. (−a)⁢x = −(a⁢x) = a⁢(−x) for each a ∈ F and for each x ∈ V
  3. a⁢0 = 0 for each a ∈ F

Notation (n-tuple): Let a1, a2, …, an be elements of a field F. We call an object of the following form an n-tuple with elements from F:

(a1, a2, …, an)
where the following statement holds:
(a1, a2, …, an) = (b1, b2, …, bn) ⇔ ai = bi (for any i, where 1 ≤ i ≤ n)
Recall that two sets A and B are called equal, written A = B, if they contain exactly the same elements. For example,
{1, 2, 3} = {2, 1, 3}
Note that with the n-tuple, the order of elements matters:
(1, 2, 3) ≠ (2, 1, 3)

Notation (a set of n-tuples): The set of all n-tuples with entries from a filed F is denoted Fn, e.g.,

Let u be an n-tuple (a1, a2, …, an), then u ∈ Fn
Let v be an n-tuple (b1, b2, …, bn), then v ∈ Fn
Fn is a vector space over F with the operations of coordinative addition and scalar multiplication; that is, if c ∈ F, then
u+v = (a1+b1, a2+b2, …, an+bn)
cu = (ca1, ca2, …, can)

Notation (column vector): Vectors in Fn may be written as column vectors:

( a1 a2 ⋮ an )
rather than as row vectors ( a1, a2, …, an )

Notation (matrix): An m×n matrix with entries from a field F is

( a11 a12 ⋯ a1n a21 a22 ⋯ a2n ⋮ ⋮ ⋮ am1 am2 ⋯ amn )
where each entry aij (1 ≤ i ≤ m, 1 ≤ j ≤ n) is an element of F.
  • diagonal entries: aij with i=j
  • ith row: ai1, ai2, …, ain
  • jth column: a1j, a2j, …, amj
  • zero matrix, denoted Ø
  • square matrix, if m=n

We denote

  • matrices by capital letters, e.g. A, and
  • the entry of a matrix A that lies in row i and column j by Aij.

Notation (a set of matrix): The set of all m×n matrices with each entries from a field F is denoted Mm×n(F).

Mm×n(F) is vector space over F with the following operations of matrix addition and scalar multiplication; for A, B ∈ Mm×n(F) and c ∈ F, and 1 ≤ i ≤ m and 1 ≤ j ≤ n,

( A+B ) i⁣j = A i⁣j + B i⁣j ( c ⁢ A ) i⁣j = c ⁢ A i⁣j

The definitions of matrix addition and scalar multiplication in Mm×n(F) are natural extensions of the corresponding operations in Fn and Fm. Let A, B ∈ Mm×n(F) and c ∈ F, then

  • A + B is the matrix Mm×n(F) whose ith row vector is the sum of the ith row vectors of A and B, and
  • c⁢A is the matrix Mm×n(F) whose ith row vector is c times the ith row vector of A.
