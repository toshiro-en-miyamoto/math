Linear Algebra

Fields (Appendix C)

Definition (Field): A field is a set on which two operations + (addition) and · (multiplication) are defined so that, for each pair of elements x, y in F, there are unique elements in F, denoted x+y and x·y, and such that the following conditions hold for all elements a, b, c in F.

The Field Axioms
addition multiplication
commutativity a+b = b+a a·b = b·a
associativity (a+b)+c = a+(b+c) (a·b)·c = a·(b·c)
existence of identity 0+a = a 1·a = a
existence of inverse a+(−a) = 0 a·a-1 = 1 (if a≠0)
distributivity of · over + a·(b+c) = a·b+a·c

Theorem (Cancellation Laws): Let F be a field. For all elements a, b, c ∈ F,

1. a+b = c+b ⇒ a = c
2. a·b = c·b and b≠0 ⇒ a = c
Proof:
  1. Existence of additive inverse of the field axioms guarantees that there exists an element −b ∈ F.
    (a+b)+(−b) = (c+b)+(−b) by adding (−b) to both sides
    a+{b+(−b)} = c+{b+(−b)} by associativity of addition
    a+0 = c+0 by existence of additive inverse
    a = c by existence of additive identity
  2. Existence of multiplicative inverse of the field axioms guarantees that if b≠0, there exists an element b-1 ∈ F. Then, the proof is similar to 1.

Corollary (Unique identities and inverses): Let F be a field and a ∈ F, then

  • Additive identity (0) is unique.
  • Additive inverse (−a) is unique.
  • Multiplicative identity (1) is unique.
  • Multiplicative inverse (a-1 if a≠0) is unique.
Proof:
  • Additive identity (0): Suppose that 0′ ∈ F is an additive identity. For all a ∈ F,
    0′ + a = a since 0′ is an additive identity
    0′ + a = a = 0 + a since 0 is an additive identity
    0′ = 0 because of the additive cancellation law
  • Other three proofs are similar.

Subtraction (−) and division (÷ or ⁄) can be defined in terms of addition and multiplication by using the additive and multiplicative inverses. For all a ∈ F and all b≠0 ∈ F,

  • a−b = a+(−b)
  • a÷b = a⁄b = a·b-1
