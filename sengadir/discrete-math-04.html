4.1 A Fundamental Counting Principle

Example 4.6

How many seven-letter words can be formed using the letters A, B, C, D and E if

  1. the word must contain exactly one D?
  2. the word must contain at least one D?

Solution:

  1. We can allot D to any of the seven places and the remaining six places can be filled by the remaining four letters in 46 ways. Hence, the required number of words is 7 × 46 .
  2. The total number of words of length seven, which can be formed by using the given five letters is 57. The number of words of length seven without D is the number of words of length seven, which can be formed with the remaining four letters, which is 47. Hence, the number of words with at least one D is 57 − 47 .

Example 4.7

Show that the number of ways of placing n distinct objects into m distinct boxes is = m n .

Solution: The first object can go into any one of the m boxes. Clearly, there are m ways of doing this. Similarly, the second object can also be placed in any of the m boxes and so on. Hence, the required answer is m × m × … , n times = m n .

Example 4.2

Find the number of all possible six-digit numbers that can be formed by using digits 2, 6, 8 and 9.

Solution: The unit place can be filled by any of the four given digits, the ten's place can be filled by any of the given four digits and so on upto to the hundred thousand's place. Hence, the required answer is 4 × 4 × 4 × 4 × 4 × 4 = 4 6 .

Note: Let us now compare Example 4.2 and Example 4.7. In Example 4.2, if we think of the places as boxes and the numbers as objects, a wrong application of Example 4.7 gives us the answer 64. To resolve this confusion, we prove the following mathematical principle.

Proposition 4.1

Let A and B be finite sets with | A | = n and | B | = m . The number of all functions from A into B is m n .

Proof: Let A = { x1 , x2 , … , xn } . f ( x1 ) can be defined in m possible ways, f ( x2 ) can be defined in m possible ways and so on. Thus, the number of all possible functions = m × m × … , n times = mn .

A function, f : A ⟼ B, is a rule that associates each element x ∈ A with the one and only element in B, denoted f(x). Note that every single element in A must be associated with an element in B.

Example 4.7 with Proposition 4.1

Show that the number of ways of placing n distinct objects into m distinct boxes is = m n .

Solution with Proposition 4.1: Consider one specific distribution of the n objects into m distinct boxes. This specific distribution defines a function f from the set of n objects into the set of boxes as follows: f ⁡ ( x ) is the box into which object x has been placed. Thus, the required number is nothing but the number functions from an n set into an m set. Hence, it is mn.

Example 4.2 with Proposition 4.1

Find the number of all possible six-digit numbers that can be formed by using digits 2, 6, 8 and 9.

Solution with Proposition 4.1: Note that a digit can occupy more than one place. Thus, we cannot define f ⁡ ( x ) as the place taken by x.

Also note that with some numbers, digits are not associated with any place. Recall that with a function f : A ⟼ B, every single element in A must be associated with an element in B.

So we take A as the set of all places and B = { 2 , 6 , 8 , 9 } . For a specific six-digit number, we define a function f : A → B as follows: For a place x, f ⁡ ( x ) is the number from the set B, which goes into x. Thus, the required answer is the number of functions from a six-element set into a four-element set, which is 46.

Note that in our description of the term function, we did not demand that every element of B to be associated with some element of A (Example 2.79).

Definition 4.1

Let there be n objects.

  • The number of ways of choosing r objects out of n objects is called the combination of r objects out of n objects. This is denoted by C r n .
  • The number of ways of choosing and arranging r objects out of the n objects is called the permutation of r objects out of n and is denoted by P r n .

Proposition 4.3

P r n = n ⁢ ( n − 1 ) ⁢ ( n − 2 ) ⁢ … ⁢ ( n − r + 1 ) = n! ( n − r ) !

P 0 n = n! ( n − 0 ) ! = 1

P 1 n = n

P n − 1 n = n ⁢ ( n − 1 ) ⁢ ( n − 2 ) ⁢ … ⁢ 2 = n!

P n n = n ⁢ ( n − 1 ) ⁢ ( n − 2 ) ⁢ … ⁢ 2 × 1 = n!

4.2.1 Circular Permutation

Proposition 4.5

The number of circular permutations of n objects is ( n − 1 ) ! .

Proof: A single circular permutation is represented by n circular arrangements. As there are n objects and n places to be filled, there are n! different circular arrangements. Thus, the number of circular permutations is n! ÷ n = ( n − 1 ) ! .

4.2.2 Permutation of Identical Objects

Illustrative Example

Imagine a child playing with cards with numerals on them. Let us say she picks up two cards of 2, four cards of 5 and nine cards of 6. Now these 15 cards can actually be arranged in 15! different ways. But among them, many of them would represent the same figures. Each figure will be counted 2! × 4! × 9! times. Hence, the number of distinct figures that can be formed is 15! 2! × 4! × 9! .

Proposition 4.6

In a collection there are m1 identical articles of type 1, m1 articles of type 2 and so on upto mk articles of type k. The number of permutations of this collection is ( m1 + m2 + … + mk ) ! m1! × m2! × … × mk!

4.3 Applications of Permutations and Combinations

Illustrative Example: Consider the expansion ( x + y + z ) 10 . The terms in this expansion look like xi ⁢ yj ⁢ zk with i + j + k = 10 . Take i=5, j=2 and k=3. The term x5 ⁢ y2 ⁢ z3 occurs, for example, as x ⁢ y ⁢ x ⁢ x ⁢ x ⁢ y ⁢ z ⁢ x ⁢ z ⁢ z , x ⁢ x ⁢ x ⁢ x ⁢ x ⁢ y ⁢ z ⁢ y ⁢ z ⁢ z and y ⁢ x ⁢ y ⁢ x ⁢ z ⁢ x ⁢ z ⁢ x ⁢ z ⁢ x etc. Thus, the number of times the term x5 ⁢ y2 ⁢ z3 occurs is the number of permutations of 10 objects of 3 kinds, each of them appearing 5, 2 and 3 times. Hence it is 10 ! 5 ! ⁢ 2 ! ⁢ 3 ! . This is the coefficient of x5 ⁢ y2 ⁢ z3 in the expansion of ( x + y + z ) 10 .

Generalizing, we see that the the coefficient of x1 i1 ⁢ x2 i2 ⁢ … ⁢ xm im in the expansion of ( x1 + x2 + … + xm ) n is m! i1! × i2! × … × im! . So, we have the following theorem known as Multinomial Theorem.

Multinomial Theorem (Theorem 4.1)

( x1 + x2 + … + xm ) n = ∑ i1 + i2 + … + im = n ⁡ ( n! i1! ⁢ i2! ⁢ … ⁢ im! ) ⁢ x1 i1 ⁢ x2 i2 ⁢ … ⁢ xm im
