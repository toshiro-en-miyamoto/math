Cartesian Product of Sets

Let us recall that points in the xy plane are represented by pairs of numbers such as ( 3 , 4 ) , ( −2 , 1 ) .

Note that the points ( 4 , 3 ) , and ( 3 , 4 ) are different. Hence, they are called ordered pairs of numbers. A pair is a collection of two objects. Often, we would like to arrange the objects of a pair in some order. That is, we would like to specify which one is the first object and which is the second. Such a pair is called an ordered pair. If a pair consists of two objects x and y and if we take x as the first and y as the second, then the ordered pair is written as ( x , y ) .

2.2 Relations

Consider two general sets A and B. We want to consider a relationship between the elements of these two sets. Some elements of A are related to some elements of B in some manner. Let us collect all the pairs ( a , b ) such that A is related to B. Recall that ( a , b ) ∈ A × B , where A × B is the cartesian product of the sets A and B. With these ideas in mind, now we define the concept of a relation as follows.

Definition 2.11 — Relations

Let A and B be two sets. A relation R from A into B is a subset of A × B . If ( a , b ) ∈ R , we say that a is related to b and denote this fact by aRb. A relation from A into A is called a relation on A.

2.2.2 Equivalence Relations and Equivalence Classes

Consider the set L of all lines in a plane and the relation R defined as l1 R l2 if l1 is parallel to l2. We have seen that this relation is both reflexive and symmetric. Further, if l1 is parallel to l2 and l2 is parallel to l3, then we know that l1 is parallel to l3. Thus, R is also transitive. This relation is an example of an equivalence relation. Its definition is given below.

Definition 2.16 — Equivalence Relations

A relation R on a set S is called an equivalence relation if

  1. R is reflexive.
  2. R is symmetric.
  3. R is transitive.

Example 2.64

The easiest example of an equivalence relation is an equality. The relation R defined as aRb if a=b is obviously an equivalence relation.

Example 2.65

Let A be a finite set and 𝒫 ⁡ ( A ) be its power set. A relation R on 𝒫 ⁡ ( A ) is defined as XRY if X and Y have the same number of elements. Now, we show that R is an equivalence relation.

  • Reflexivie: XRX means that X and X have the same number of elements, hence it is true.
  • Symmetric: Next, XRY. This means that X and Y have the same number of elements. This can also be stated as Y and X have the same number of elements. So, YRX. Hence R is symmetric.
  • Transitive: Further, XRY and YRZ imply that X and Y have the same number of elements and Y and Z have the same number of elements. It is clear that X and Z will have the same number of elements. So, XRZ and R is transitive.
We have now proved that R is an equivalence relation.

2.3 Partitions

Definition 2.17 — Partitions

Let S be a set and { Aλ : λ∈Λ } be a collection of non-empty subsets of S. We say that this collection defines a partition of S if

  • S = ∪ λ∈Λ ⁡ Aλ
  • A λ₁ ∩ A λ₂ = ∅ if A λ₁ ≠ A λ₂

Example 2.68

Let S be the set of students of a class. Let A and B be, respectively, the set of girl students and the set of boy students of the same class. We have

  1. A ∪ B = S
  2. A ∩ B = ∅
The subsets, A and B are disjoint. This illustrates the idea of the partition of a set.

Define R on ℤ as aRb if a−b is a multiple of4 . Under this equivalence relation, 0 is related to 0, 4, 8, etc. Similarly, 1 is related to 1, 5, 9, etc. So, now let us define the following subsets of ℤ as

2.3.1 Partial Order

Definition 2.18 — Partial Orders

A relation R on a set S is called a partial order if

  • R is reflexive
  • R is anti-symmetric
  • R is transitive

Example 2.75

Let S = { 1 , 2 , 3 , … , 10 } and R is defined as aRb if a≤b . Then R is a partial order.

Solution: For a, b, and c ∈ S

  • Reflexive: For any a, we have a≤a.
  • Anti-symmetric: If a≤b and b≤a then a=b.
  • Transitive: If a≤b and b≤c then a≤c.
Thus, R is a partial order.

Counter Example: Let S = { ∅ , { 1 } , { 2 } , { 1 , 2 } } . Let n ⁡ ( A ) be the number of elements of A. Define ARB if n ⁡ ( A ) ≤ n ⁡ ( B ) . Then R is not a partial order.

  • Reflexive: n ⁡ ( A ) = n ⁡ ( A ) hence ARA for all A∈S.
  • Not anti-symmetric: Now, consider A= { 1 } and B= { 2 } . We have n ⁡ ( A ) ≤ n ⁡ ( B ) and n ⁡ ( B ) ≤ n ⁡ ( A ) , since n ⁡ ( A ) = n ⁡ ( B ) = 1 . Thus ARB and BRA. But, A≠B.
  • Transitive: If n ⁡ ( A ) ≤ n ⁡ ( B ) and n ⁡ ( B ) ≤ n ⁡ ( C ) then n ⁡ ( A ) ≤ n ⁡ ( C ) .

2.4 Functions

Definition 2.19 — Functions

A function f from a set A into a set B is a rule that assigns precisely one element of B to each element of A. A is called the domain of the function and B is called the co-domain of the function. If b∈B is the element that is assigned to a∈A, then we write f ⁡ ( a ) = b . b is called the image of a under f and a is called a pre-image of b under f.
