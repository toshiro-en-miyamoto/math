Chapter 3 Presentation Markup

3.2 Token Elements

3.2.3 Identifier <mi>

Identifiers include function names such as "sin". Expressions such as "sin x" should be written using the character U+2061 as shown below; 

  <mrow>
    <mi>sin</mi>
    <mo>&ApplyFunction;</mo>
    <mi>x</mi>
  </mrow>

Miscellaneous text that should be treated as a "term" can also be represented by an mi element, as in: 

  <mrow>
    <mn>1</mn>
    <mo>&plus;</mo>
    <mi>&hellip;</mi>
    <mo>&plus;</mo>
    <mi>n</mi>
  </mrow>

3.2.5 Operator, Fence, Separator or Accent <mo>

3.2.5.3 Operators

  <mrow>
    <mn>1</mn>
    <mo>&plus;</mo>
    <mn>2</mn>
    <mo>&equals;</mo>
    <mn>5</mn>
  </mrow>

  <mrow>
    <mo>if</mo>
    <mi>x</mi>
    <mo>is an integer</mo>
  </mrow>

3.2.5.4 Fences

  <mrow>
    <mo fence="true">&lpar;</mo>
    <mi>a</mi>
    <mo>&plus;</mo>
    <mi>b</mi>
    <mo fence="true">&rpar</mo>
  </mrow>

  <mrow>
    <mi>f</mi>
    <mo>&ApplyFunction;</mo>
    <mrow>
      <mo fence="true">&lpar;</mo>
      <mi>x</mi>
      <mo separator="true">&comma;</mo>
      <mi>y</mi>
      <mo fence="true">&rpar</mo>
    </mrow>
  </mrow>

3.2.5.5 Invisible operators

  <mrow>
    <mi>x</mi>
    <mo>&InvisibleTimes;</mo>
    <mi>y</mi>
  </mrow>

    </mrow>

  </mrow>
