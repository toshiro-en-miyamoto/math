3 Presentation Markup

3.2 Token Elements

3.2.3 Identifier <mi>

Identifiers include function names such as "sin". Expressions such as "sin x" should be written using the character U+2061 as shown below; 

<mrow>
    <mi>sin</mi>
    <mo>&ApplyFunction;</mo>
    <mi>x</mi>
</mrow>
sin ⁡ x

Miscellaneous text that should be treated as a "term" can also be represented by an mi element, as in: 

<mrow>
    <mn>1</mn>
    <mo>&plus;</mo>
    <mi>&hellip;</mi>
    <mo>+</mo>
    <mi>n</mi>
</mrow>
1 + … + n
