3 Presentation Markup

3.2 Token Elements

3.2.3 Identifier <mi>

Identifiers include function names such as "sin". Expressions such as "sin x" should be written using the character U+2061 as shown below; 

<mrow>
    <mi>sin</mi>
    <mo>&#x2061;<!--FUNCTION APPLICATION--></mo>
    <mi>x</mi>
</mrow>
sin ⁡ x

Miscellaneous text that should be treated as a "term" can also be represented by an mi element, as in: 

<mrow>
    <mn>1</mn>
    <mo>+</mo>
    <mi>&#x2026;<!--HIRIZONTAL ELLIPSIS--></mi>
    <mo>+</mo>
    <mi>n</mi>
</mrow>
1 + … + n

3.2.6.4 Mixing text and mathematics

<mrow>
    <mo>there exists</mo>
    <mrow>
        <mi>&delta;</mi>
        <mo>&gt;</mo>
        <mn>0</mn>
    </mrow>
    <mo>such that</mo>
    <mrow>
        <mrow>
            <mi>f</mi>

                <mo>(</mo>

    </mrow>

Appendix

The names of symbolic constants should be represented as mi elements:

  • Α, α, alpha (アルファ)
  • Β, β, beta (ベータ)
  • Γ, γ, gamma (ガンマ)
  • Δ, δ, delta (デルタ)
  • Ε, ε, epsilon (イプシロン)
  • Ζ, ζ, zeta (ゼータ)
  • Η, η, eta (エータ)
  • Θ, θ, theta (テータ)
  • Ι, ι, iota (イオタ)
  • Κ, κ, kappa (カッパ)
  • Λ, λ, lambda (ラムダ)
  • Μ, μ, mu (ミュー)
  • Ν, ν, nu (ニュー)
  • Ξ, ξ, xi (クシー)
  • Ο, ο, omicron (オミクロン)
  • Π, π, pi (パイ)
  • Ρ, ρ, rho (ロー)
  • Σ, σ, sigma (シグマ)
  • Τ, τ, tau (タウ)
  • Υ, υ, upsilon (ユプシロン)
  • Φ, φ, phi (ファイ)
  • Χ, χ, chi (キー)
  • Ψ, ψ, psi (プサイ)
  • Ω, ω, omega (オメガ)

The Unicode defines "Invisible operators:"

  • U+2061 &ApplyFunction;, FUNCTION APPLICATION
  • U+2062 &InvisibleTimes;, INVISIBLE TIMES
  • U+2063 &InvisibleComma;, INVISIBLE SEPARATOR
  • U+2064 (no short name), INVISIBLE PLUS

The Unicode defines "Letterlike symbols:"

  • U+2102 ℂ, &complexes;, the set of complex numbers (複素数)
  • U+2115 ℕ, &naturals;, the set of natural numbers (自然数)
  • U+2119 ℙ, &primes;, the set of primes (素数)
  • U+211A ℚ, &rationals;, the set of rational numbers (有理数)
  • U+211D ℝ, &reals;, the set of real numbers (実数)
  • U+2124 ℤ, &integers;, the set of integers (整数)

  • U+00AC ¬, &not;, NOT SIGN (否定)
  • U+00B1 ±, &PlusMinus;, PLUS-MINUS SIGN
  • U+00D7 ×, &times;, MULTIPLICATION SIGN (乗算)
