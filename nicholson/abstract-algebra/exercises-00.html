Chapter 0: Preliminaries

0.1 Proofs

The most common methods of proof include:

  • Direct Proof
  • Reduction to Cases
  • Proof by Contradiction

    • 1. In each case, prove the result and either prove the converse or give a counterexample.

    a. If n is an even integer, then n2 is a multiple of 4.

    Solution to a. If n is an even integer, it has the form n = 2 ⁢ k for some integer k. Then n2 = 4 ⁢ k2 is a multiple of 4 because k2 is an integer. ∎ (Direct Proof)

    The converse of a. If n2 is a multiple of 4 then n is an even integer.

    Proving the converse. Suppose n is not an even integer so that n is an odd, say n = 2 ⁢ k + 1 , where k is an integer. Then

    n2 = 2 ⁢ k + 1 2 = 4 ⁢ k2 + 4 ⁢ k + 1 = 4 ⁢ k2 + k + 1

    As k2 + k is an integer, n2 is not a multiple of 4, a contradiction. ∎ (Proof by Contradiction)

    b. If m is an even integer and n is an odd integer, then m+n is odd.

    Solution to b. If m is an even integer, say m = 2 ⁢ j for some integer j, and n is an odd integer, say n = 2 ⁢ k + 1 for some integer k, then m + n = 2 ⁢ j + 2 ⁢ k + 1 = 2 ⁢ j + k + 1 is odd. ∎ (Direct Proof)

    The converse of b. If m+n is odd, then m is an even integer and n is an odd integer.

    Proving the converse. With the same way as above, the following cases are proved:

    # m n m+n
    (ⅰ) even even even
    (ⅱ) even odd odd
    (ⅲ) odd even odd
    (ⅳ) odd odd even
    Hence, from the case (ⅱ), the converse is true. ∎ (Reduction to Cases)

    c. If x=2 or x=3, then x3 − 6 ⁢ x2 + 11 ⁢ x − 6 = 0

    Solution to c. As x3 − 6 ⁢ x2 + 11 ⁢ x − 6 = x − 1 ⁢ x − 2 ⁢ x − 3 , if x=2 then 1 × 0 × − 1 = 0 , and if x=3 then 2 × 1 × 0 = 0 . ∎ (Direct Proof)
