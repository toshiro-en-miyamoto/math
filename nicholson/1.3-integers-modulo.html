1.3 Integers Modulo n

Congruent Modulo n

Let n ≥ 2 be an integer. Then integers a and b are said to be congruent modulo n provided that n ∣ ( a − b ) . In this case we write

and refer to n as the modulus.

Thus, we have 2 ≡ 5 ⁡ ( mod ⁡ 3 ) , 21 ≡ 16 ⁡ ( mod ⁡ 5 ) , and −4 ≡ 2 ⁡ ( mod ⁡ 6 ) . Note that a ≡ 0 ⁡ ( mod ⁡ n ) if and only if n ∣ a . We assume that n ≥ 2 because congruence modulo 0 or 1 is of no interest.

Theorem 1

Congruence modulo n is an equivalence on ℤ; that is:

  1. a ≡ a ⁡ ( mod ⁡ n ) for every integer a.
  2. a ≡ b ⁡ ( mod ⁡ n ) ⇒ b ≡ a ⁡ ( mod ⁡ n )
  3. a ≡ b ⁡ ( mod ⁡ n ) ∧ b ≡ c ⁡ ( mod ⁡ n ) ⇒ a ≡ c ⁡ ( mod ⁡ n )

If a is an integer, its equivalence class [ a ] with respect to congruence modulo n is called its residue class modulo n, and we write

For example, if n = 2 ,

  • 0 ‾ = { x ∈ ℤ | x ≡ 0 ⁡ ( mod ⁡ 2 ) } = the set of even integers
  • 1 ‾ = { x ∈ ℤ | x ≡ 1 ⁡ ( mod ⁡ 2 ) } = the set of odd integers

Theorem 2-1

Given n ≥ 2 , a ‾ = b ‾ if and only if a ≡ b ⁡ ( mod ⁡ n ) .

Proof.

  • Suppose a ‾ = b ‾ . Since a ∈ a ‾ , we have a ∈ b ‾ , so a ≡ b .
  • Conversely, let a ≡ b . Since a ‾ and b ‾ are sets, we must show that a ‾ ⊆ b ‾ and b ‾ ⊆ a ‾ . If x ∈ a ‾ , then x ≡ a ; so, as a ≡ b , we have x ≡ b by (3) of Theorem 1. This proves that a ‾ ⊆ b ‾ . Since b ≡ a by (2) of Theorem 1, a similar proof shows that b ‾ ⊆ a ‾ . ▮

Theorem 3

Let n ≥ 2 be an integer.

  1. If a ∈ ℤ , then a ‾ = r ‾ for some r where 0 ≤ r ≤ n − 1 .
  2. The residue classes 0 ‾ , 1 ‾ , 2 ‾ , … , n − 1 ‾ modulo n are distinct.

Proof.

  1. The division algorithm gives a = q ⁢ n + r , where 0 ≤ r ≤ n − 1 , so a ≡ r ⁡ ( mod ⁡ n ) . ▮
  2. Suppose r ‾ = s ‾ where 0 ≤ r ≤ n − 1 and 0 ≤ s ≤ n − 1 . We may assume that r ≤ s . Then r ‾ = s ‾ means that r ≡ s ⁡ ( mod ⁡ n ) , so s − r is an integral multiple of n such that 0 ≤ s − r ≤ n − 1 . This implies that r = s . ▮

Theorem 2-2

Congruence modulo n is compatible with addition and multiplication of integers in the following sense. Let a , a 1 , b , and b 1 denote integers. If

  • a ≡ a 1 ⁡ ( mod ⁡ n )
  • b ≡ b 1 ⁡ ( mod ⁡ n )

then

  • a + b ≡ a 1 + b 1 ⁡ ( mod ⁡ n )
  • a · b ≡ a 1 · b 1 ⁡ ( mod ⁡ n )

Proof. Let a − a 1 = p ⁢ n and b − b 1 = q ⁢ n , where p and q are integers. Adding these equations gives ( a + b ) − ( a 1 + b 1 ) = ( p + q ) ⁢ n , and this implies that a + b ≡ a 1 + b 1 ⁡ ( mod ⁡ n ) . Similarly, multiplying the equations a = a 1 + p ⁢ n and b = b 1 + q ⁢ n gives a · b − a 1 · b 1 = ( b 1 ⁢ p + a 1 ⁢ q + p ⁢ q ⁢ n ) ⁢ n , this implies that a · b ≡ a 1 · b 1 ⁡ ( mod ⁡ n ) . ▮

Addition and Multiplication mod n

We define addition and multiplication of residue classes a ‾ and b ‾ in ℤ n by

  • a ‾ + b ‾ = a + b ‾
  • a ‾ · b ‾ = a · b ‾

Proof of well-definedness. Suppose a ‾ = a 1 ‾ and b ‾ = b 1 ‾ where a ≠ a 1 and b ≠ b 1 are possible. If we add (multiply) these classes as a ‾ and b ‾ , the definition of addition and multiplication mod n gives their sum as a + b ‾ and product as a · b ‾ , but if we represent the classes as a 1 ‾ and b 1 ‾ , their sum is a 1 + b 1 ‾ and product is a 1 · b 1 ‾ . Clearly, the definitions of addition and multiplicaion make no sense unless

  • a + b ‾ = a 1 + b 1 ‾
  • a · b ‾ = a 1 · b 1 ‾

Again, suppose

  • a ‾ = a 1 ‾
  • b ‾ = b 1 ‾

By Theorem 2-1; a ‾ = a 1 ‾ ⇔ a ≡ a 1 ⁡ ( mod ⁡ n ) ,

  • a ≡ a 1 ⁡ ( mod ⁡ n )
  • b ≡ b 1 ⁡ ( mod ⁡ n )

Then Theorem 2-2 gives;

  • a + b ≡ a 1 + b 1 ⁡ ( mod ⁡ n )
  • a · b ≡ a 1 · b 1 ⁡ ( mod ⁡ n )

Thus, again by Theorem 2-1, a + b ‾ = a 1 + b 1 ‾ and a · b ‾ = a 1 · b 1 ‾ as required. ▮

Example 4. In ℤ 6 compute 3 ‾ + 5 ‾ and 3 ‾ · 5 ‾ .

Solution. The definition gives 3 ‾ + 5 ‾ = 8 ‾ = 2 ‾ , because 8 ≡ 2 ⁡ ( mod ⁡ 6 ) . Similarly, 3 ‾ · 5 ‾ = 15 ‾ = 3 ‾ .

Theorem 5

Let a and n be integers with n ≥ 2 . Then a ‾ has an inverse in ℤ n if and only if a and n are relatively prime. If a ‾ has an inverse, that inverse is unique.

Proof. If a and n are relatively prime, then 1 = gcd ⁡ ( a , n ) is a linear combination of a and n (by Theorem 4 1.2), say 1 = b ⁢ a + c ⁢ n , where b and c are integers. Hence, b ⁢ a ≡ 1 ⁡ ( mod ⁡ n ) , so b ‾ · a ‾ = 1 ‾ by Theorem 2. Conversely, if b exists such that b ‾ · a ‾ = 1 ‾ then b ⁢ a ≡ 1 ⁡ ( mod ⁡ n ) . Thus, n ∣ ( 1 − b ⁢ a ) , say 1 − b ⁢ a = q ⁢ n for some integer q. But then 1 = b ⁢ a + q ⁢ n , so a and n are relatively prime (again by Theorem 4 1.2). ▮

Proof of uniqueness. Let a ‾ , b ‾ , and c ‾ be residue classes in ℤ n . If b ‾ · a ‾ = 1 ‾ and c ‾ · a ‾ = 1 ‾ , then multiply c ‾ · a ‾ = 1 ‾ by b ‾ to get c ‾ · a ‾ · b ‾ = b ‾ , that is c ‾ · ( a ‾ · b ‾ ) = c ‾ = b ‾ . ▮

Theorem 6. Chinese Remainder Theorem ‐ two congruences

Let m and n be relatively prime integers. If s and t are arbitrary integers, there exists a solution x ∈ ℤ to the simultaneous congruences

  • x ≡ s ⁡ ( mod ⁡ m )
  • x ≡ t ⁡ ( mod ⁡ n )

Proof. Since gcd ⁡ ( m , n ) = 1 , the euclidean algorithm gives p and q in ℤ such that 1 = m ⁢ p + n ⁢ q . Take

Then, x ≡ s ⁡ ( mod ⁡ m ) , because x − s = m ⁢ p ⁢ t + ( 1 − m ⁢ p ) ⁢ s − s = m ⁢ p ⁢ ( t − s ) . A similar argument gives x ≡ t ⁡ ( mod ⁡ n ) . ▮

Theorem 6. Chinese Remainder Theorem ‐ three congruences

The previous method can be iterated to solve a system of more than two congruences, provided that only the moduli are relatively prime in pairs.

Let m 1 , m 2 , and m 3 be integers relatively prime in pairs. Given arbitrary integers s 1 , s 2 , and s 3 , we want to find an integer x such that

  • x ≡ s 1 ⁡ ( mod ⁡ m 1 )
  • x ≡ s 2 ⁡ ( mod ⁡ m 2 )
  • x ≡ s 3 ⁡ ( mod ⁡ m 3 )

Solution. Firstly, apply the Chinese remainder theorem to obtain x 2 such that

  • x 2 ≡ s 1 ⁡ ( mod ⁡ m 1 )
  • x 2 ≡ s 2 ⁡ ( mod ⁡ m 2 )

Then, since m 1 · m 2 and m 3 are relatively prime, apply the Chinese remainder theorem again to obtain x such that

  • x ≡ x 2 ⁡ ( mod ⁡ m 1 · m 2 )
  • x ≡ s 3 ⁡ ( mod ⁡ m 3 )

Proof. x ≡ x 2 ⁡ ( mod ⁡ m 1 ) because x − x 2 = p · m 1 · m 2 = q · m 1 for some integers p and q, so since x 2 ≡ s 1 ⁡ ( mod ⁡ m 1 ) , we have x ≡ s 1 ⁡ ( mod ⁡ m 1 ) by the transitive property of the congruence modulo n. Similarly, x ≡ s 2 ⁡ ( mod ⁡ m 2 ) . ▮

Theorem 7

The following are equivalent for an integer n ≥ 2 .

  1. Every element a ‾ ≠ 0 ‾ in ℤ n has an inverse.
  2. If a ‾ · b ‾ = 0 ‾ in ℤ n then either a ‾ = 0 ‾ or b ‾ = 0 ‾ .
  3. n is a prime.

Hence, if p is a prime, ℤ p has the property that every nonzero element has an inverse. This is also true of the real numbers ℝ, and such systems are called fields.

Corollary (Exercise 26). If p is a prime and x 2 = a ‾ 2 in ℤ p , then x = a ‾ or x = − a ‾ .

Proof. Let b ‾ ≠ 0 ‾ and c ‾ ≠ 0 ‾ in ℤ p , then b ‾ 2 = c ‾ 2 implies b ‾ 2 − c ‾ 2 = ( b ‾ − c ‾ ) · ( b ‾ + c ‾ ) = 0 ‾ . So b ‾ − c ‾ = 0 ‾ or b ‾ + c ‾ = 0 ‾ by Theorem 7; that is, b ‾ = c ‾ or b ‾ = − c ‾ . ▮

Corollary: Self-inverses. If p is a prime, only 1 ‾ and p − 1 ‾ are self-inverse elements in ℤ p .

Proof. If p is a prime and x 2 = 1 ‾ 2 = 1 ‾ in ℤ p , then Theorem 7 gives x = 1 ‾ or x = − 1 ‾ = p − 1 ‾ (Exercise 35). As each invese element is unique in ℤ p by Theorem 5, only 1 ‾ and p − 1 ‾ are self-inverse elements in ℤ p . ▮

Corollary: Wilson's Theorem. If p is a prime, then ( p − 1 ) ! ≡ − 1 ⁡ ( mod ⁡ n ) .

Proof. Since p is prime, each element 1 ‾ , 2 ‾ , 3 ‾ , … , p − 1 ‾ in ℤ p has an inverse by Theorem 7. Hence, pairs of inverses in the product p − 1 ‾ ! = 1 ‾ · 2 ‾ · 3 ‾ · … · p − 1 ‾ will cancel leaving only the self-inverse elements 1 ‾ and − 1 ‾ (Exercise 26). Thus, p − 1 ‾ ! = 1 ‾ · − 1 ‾ = − 1 ‾ in ℤ p as required. ▮
