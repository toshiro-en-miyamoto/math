0.1 Proofs

The most common methods of proof include:

  • Direct Proof
  • Reduction to Cases
  • Proof by Contradiction

    • Exercise 0.1-1

    In each case, prove the result and either prove the converse or give a counterexample.

    0.1-1-(a)

    Implication
    If n is an even integer, then n2 is a multiple of 4.
    Solution to the implication
    If n is an even integer, it has the form n = 2 ⁢ k for some integer k. Then n2 = 4 ⁢ k2 is a multiple of 4 because k2 is an integer. ▮ (Direct Proof)
    The converse
    If n2 is a multiple of 4 then n is an even integer.
    Solution to the converse
    The converse is true. Suppose n is not an even integer so that n is an odd, say n = 2 ⁢ k + 1 , where k is an integer. Then n2 = 2 ⁢ k + 1 2 = 4 ⁢ k2 + 4 ⁢ k + 1 = 4 ⁢ k2 + k + 1 . As k2 + k is an integer, n2 is not a multiple of 4, a contradiction. ▮ (Proof by Contradiction)

    0.1-1-(b)

    Implication
    If m is an even integer and n is an odd integer, then m+n is odd.
    Solution to the implication
    If m is an even integer, say m = 2 ⁢ j for some integer j, and n is an odd integer, say n = 2 ⁢ k + 1 for some integer k, then m + n = 2 ⁢ j + 2 ⁢ k + 1 = 2 ⁢ j + k + 1 is odd. ▮ (Direct Proof)
    Converse
    If m+n is odd, then m is an even integer and n is an odd integer.
    Solution to the converse
    The converse is true. With the same way as above, the following cases are proved:
    # m n m+n
    (ⅰ) even even even
    (ⅱ) even odd odd
    (ⅲ) odd even odd
    (ⅳ) odd odd even
    Hence, from the case (ⅱ), the converse is true. ▮ (Reduction to Cases)

    0.1-1-(c)

    Implication
    If x=2 or x=3, then x3 − 6 ⁢ x2 + 11 ⁢ x − 6 = 0
    Solution to the implication
    As x3 − 6 ⁢ x2 + 11 ⁢ x − 6 = x − 1 ⁢ x − 2 ⁢ x − 3 , if x=2 then 1 × 0 × − 1 = 0 , and if x=3 then 2 × 1 × 0 = 0 . ▮ (Direct Proof)
    Converse
    If x3 − 6 ⁢ x2 + 11 ⁢ x − 6 = 0 , then x=2 or x=3.
    Solution to the converse
    The converse is false. As x3 − 6 ⁢ x2 + 11 ⁢ x − 6 = x − 1 ⁢ x − 2 ⁢ x − 3 = 0 , x − 1 = 0 or x − 2 = 0 or x − 3 = 0 . Hence, not only x=2 or x=3 but also x=1. ▮ (Counterexample)

    0.1-1-(d)

    Implication
    If x2 − 5 ⁢ x + 6 = 0 , then x=2 or x=3.
    Solution to the implication
    As x2 − 5 ⁢ x + 6 = x − 2 ⁢ x − 3 = 0 , x − 2 = 0 or x − 3 = 0 . Hence, x=2 or x=3. ▮ (Direct Proof)
    Converse
    If x=2 or x=3. then x2 − 5 ⁢ x + 6 = 0 .
    Solution to the converse
    The converse is true. As x2 − 5 ⁢ x + 6 = x − 2 ⁢ x − 3 , if x=2 then 0 × − 1 = 0 , and if x=3 then 1 × 0 = 0 . ▮ (Direct Proof)

    Exercise 0.1-2

    In each case, prove the result by splitting into cases or give a counterexample.

    0.1-2-(a)

    Implication
    If n is any integer, then n2 = 4 ⁢ k + 1 for some integer k.
    Solution to the implication
    The Implication is false. When n=2, 4 = 4 ⁢ k + 1 , hence k = 3 ÷ 4 is a fraction. ▮ (Counterexample)

    0.1-2-(b)

    Implication
    If n is any odd integer, then n2 = 8 ⁢ k + 1 for some integer k.
    Solution to the implication
    The Implication is true. As n is an odd integer, say n = 2 ⁢ j + 1 for some integer j, n2 = 2 ⁢ j + 1 2 = 4 ⁢ j2 + 4 ⁢ j + 1 = 8 ⁢ k + 1 , hence j ⁢ j + 1 = 2 ⁢ k , which is true because: if n is even then j ⁢ j + 1 is even; if n is odd then n+1 is even so j ⁢ j + 1 is even as well. ▮ (Reduction to Cases)

    0.1-2-(c)

    Implication
    If n is any integer, then n3 − n = 3 ⁢ k for some integer k.
    Solution to the implication
    The Implication is true. As n3 − n = n ⁢ n − 1 ⁢ n + 1 = 3 ⁢ k , the Implication is true because: if n ⁡ mod ⁡ 3 = 0 then n ⁢ n − 1 ⁢ n + 1 is a multiple of 3; if n ⁡ mod ⁡ 3 = 1 then n − 1 ⁡ mod ⁡ 3 = 0 so n ⁢ n − 1 ⁢ n + 1 is a multiple of 3; and if n ⁡ mod ⁡ 3 = 2 then n + 1 ⁡ mod ⁡ 3 = 0 so n ⁢ n − 1 ⁢ n + 1 is a multiple of 3. ▮ (Reduction to Cases)

    Exercise 0.1-3

    In each case, prove the result by contradiction and either prove the converse or give a counterexample.

    0.1-3-(a)

    Implication
    If n>2 is a prime integer, then n is odd.
    Solution to the implication
    The Implication is true. If we assume that n is even, n is a multiple of 2 and not any prime integer, a contradiction. ▮ (Proof by Contradiction)
    Converse
    If n is odd, then n>2 is a prime integer.
    Solution to the converse
    The converse is false. n=9 is odd but not a prime: 9 = 3 × 3 . ▮ (Counterexample)

    0.1-3-(b)

    Implication
    If n+m=25, where n and m are integer, one of n and m is greater than 12.
    Solution to the implication
    The Implication is true. If we assume that n≤12 and m≤12 — the negation of "one of n and m is greater than 12," then n+m ≤ 24 , a contradiction. ▮ (Proof by Contradiction)
    Converse
    If one of n and m, where n and m are integer, is greater than 12, then n+m=25.
    Solution to the converse
    The converse is false. 13+1 =14 ≠25 . ▮

    0.1-3-(c)

    Implication
    If a and b are positive numbers and a ≤ b , then a ≤ b .
    Solution to the imlication
    The Implication is true. Assume a − b > 0 . As a + b > 0 where a and b are positive numbers, a − b ⁢ a + b = a − b > 0 , a contradiction. ▮ (Proof by Contradiction)
    Converse
    If a and b are positive numbers and a ≤ b , then a ≤ b .
    Solution to the converse
    The converse is true. As b + a ≥ 0 for any positive integers, b − a ⁢ b + a = b − a ≥ 0 , hence a ≤ b . ▮ (Direct Proof)

    0.1-3-(d)

    Implication
    If m and n are integers and m⁢n is even, then m is even or n is even.
    Solution to the implication
    The Implication is true. If we assume that some integers m and n are odd, m⁢n = 2 ⁢ j + 1 ⁢ 2 ⁢ k + 1 = 2 ⁢ 2 ⁢ j ⁢ k + j + k + 1 is odd, a contradiction. ▮ (Proof by Contradiction)
    Converse
    If m and n are integers and m is even or n is even, then m⁢n is even.
    Solution to the converse
    The converse is true. If m is even then m⁢n has the form 2 ⁢ j ⁢ n , and if n is even then m⁢n has the form 2 ⁢ j ⁢ m . ▮ (Direct Proof)

    Exercise 0.1-4

    Prove each implication by contradiction.

    0.1-4-(a)

    Implication
    If x and y are positive numbers, then x+y ≠ x + y
    Solution to the implication
    If we assume x+y = x + y , then Hence x ⁢ y = 0 , which means x=0 or y=0, a contrdiction. ▮ (Proof by Contrdiction)

    0.1-4-(b)

    Implication
    If x is irrational and y is rational, then x+y is irrational.
    Solution to the implication
    If we assume x+y is rational having the form of nm where n and m are some integers, then x = n − y ⁢ m m . A contradiction because x has the form of a fraction. ▮ (Proof by Contrdiction)

    0.1-4-(c)

    Implication
    If 13 people are selected, at least 2 have birthdays in the same month.
    Solution to the implication
    Assume all have birthdays in different months, then there can be at most 12 people, one for each month, a contradiction. ▮ (Proof by Contrdiction)

    0.1-4-(d)

    Implication
    If n+1 pigeons are placed in n holes, some hole contains at least two pigeons.
    Solution to the implication
    Suppose each hole contains at most one pigeon, then the total number of pigeons is at most n, a contradiction. ▮ (Proof by Contrdiction)

    0.1-5-(a)

    Statement
    n2 + n + 11 is a prime for all positive integers n.
    Solution to the statement
    n2 + n + 11 = n ⁢ n + 1 + 11 , hence if n is a multiple of 11, say 11, then 11 × 12 + 11 = 11 × 13 . Likewise if n+1 is a multiple of 11, say n=10 then 10 × 11 + 11 = 11 × 11 . ▮ (Counterexample)

    0.1-5-(c)

    Statement
    If n points are arranged on a circle in such a way that no three of the lines joining them have a common point, these lines divide the circle into 2 n − 1 regions. For example, if n=4, there are 8 = 23 regions.
    Solution to the statement
    n=6 is a counterexample. There are then 31 regions.
