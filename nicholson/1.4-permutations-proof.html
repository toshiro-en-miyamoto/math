Proofs of 1.4 Permutations

Lemma 1. If k ∈ M σ , then σ ⁡ ( k ) ∈ M σ .

Proof. Otherwise, σ ⁡ ( k ) is fixed by σ; that is, σ ⁡ ( σ ⁡ ( k ) ) = σ ⁡ ( k ) . But then the fact that σ is one-to-one gives σ ⁡ ( k ) = k (recall that if a mapping f is one-to-one, f ⁡ ( a ) = f ⁡ ( b ) ⇒ a = b ), which is contrary to the hypothesis k ∈ M σ . ▮

Two permutations σ and τ are called disjoint if no element of Xn is moved by both; that is, if M σ ∩ M τ = ∅ .

Theorem 3. If σ and τ in S n are disjoint, then σ ∘ τ = τ ∘ σ .

Proof. For k ∈ X n , we must show that τ ∘ σ ( k ) = σ ∘ τ ( k ) . Since M σ ∩ M τ = ∅ by hypothesis, there are three cases:

  1. k ∈ M σ : Then σ ⁡ ( k ) ∈ M σ too (by Lemma 1), so neither lies in M τ . Hence, both are fixed by τ, so τ ⁡ ( k ) = k and τ ⁡ ( σ ⁡ ( k ) ) = σ ⁡ ( k ) . Hence, τ ∘ σ ( k ) = τ ⁡ ( σ ⁡ ( k ) ) = σ ⁡ ( k ) = σ ⁡ ( τ ⁡ ( k ) ) = σ ∘ τ ( k )
  2. k ∈ M τ : This case is analogous to Case 1.
  3. k ∉ M σ and k ∉ M τ : Then σ ⁡ ( k ) = k and τ ⁡ ( k ) = k , so τ ∘ σ ( k ) = τ ⁡ ( σ ⁡ ( k ) ) = σ ⁡ ( k ) = σ ⁡ ( τ ⁡ ( k ) ) = σ ∘ τ ( k ) . ▮
