6. If σ and τ fix k, show that σ ∘ τ and σ − 1 both fix k.

Solution. If σ fixes k, then we have σ ⁡ ( k ) = k .

  1. If τ fixes k, we get τ ⁡ ( k ) = k , so σ ⁡ ( τ ⁡ ( k ) ) = k . Hence, σ ∘ τ ⁡ ( k ) = k , meaning that σ ∘ τ fixes k.
  2. Suppose, on the contrary, that σ − 1 moves k, say σ − 1 ⁡ ( j ) = k where j ≠ k . Then, we get σ ⁡ ( σ − 1 ⁡ ( j ) ) = k , therefore σ ∘ σ − 1 ⁡ ( j ) = k , a contradiction to the definition of the inverse σ ∘ σ − 1 ⁡ ( j ) = ε ⁡ ( j ) = j . The similar argument goes with σ − 1 ⁡ ( k ) = i where k ≠ i , leading to σ − 1 ∘ σ ⁡ ( k ) = i , a contradiction to σ − 1 ∘ σ ⁡ ( k ) = ε ⁡ ( k ) = k .

  • Then we can say that M σ = ∅ and M τ = ∅ , meaning that σ = ε and τ = ε .
