Example 1

Equality is an equivalence on any set A. If a∈A, the equivalence class of a is [ a ] = { x∈A | x=a } = { a } , the singleton.

Solution 1.

  1. Reflexive: For all a∈A, a=a . Thus a ≡ a .
  2. Symmetric: If a≡b , then a=b , which means b=a , hence b≡a .
  3. Transitive: If a≡b and b≡c , then a =b =c , so a=c , hence a≡c .
Suppose the equivalence class [ a ] has two elements, say [ a ] = { a , a1 } , then a ≡ a1 , so a = a1 . Hence [ a ] = { a , a } = { a } , the singleton.

Example 2

Being parallel is an equivalence on the set of lines in the plane. The equivalence class of a given line consists of all lines parallel to it.

Solution 2.

  1. Reflexive: Given a line l in the plane, the line is parallel to itself, hence l≡l.
  2. Symmetric: Let l and l1 be lines in the plane. If l ≡ l1 , then l ∥ l1 so l1 ∥ l . Hence l1 ≡ l .
  3. Transitive: Let l, l1, and l2 be lines in the plane. If l ≡ l1 and l1 ≡ l2 , then l ∥ l1 ∥ l2 . Hence l ≡ l2 .

Example 3

If X and Y are subsets of a finite set U, write X≡Y to mean | X | = | Y | . Then ≡ is an equivalence on the set of subsets of U, and [ X ] consists of all subsets with the same number of elements as X.

Solution 3.

  1. Reflexive: Given a finite set X, | X | = | X | . Thus X ≡ X .
  2. Symmetric: If X ≡ Y , then | X | = | Y | , so | Y | = | X | . Thus Y ≡ X .
  3. Transitive: Let X, Y, and Z be subsets of U. If X ≡ Y and Y ≡ Z , then | X | = | Y | = | Z | . Hence X ≡ Z .
Let | X | = n , then [ X ] = { A is a subset of U | A ≡ X } = { A is a subset of U | A has n elements } , thus [ X ] contains all subsets with the same number of elements as X.

Example 4

Let α : A → B be a mapping. If a and a1 are elements of A, write a ≡ a1 to mean α ⁡ ( a ) = α ⁡ ( a 1 ) . Then ≡ is an equivalence on A, called the kernel equivalence of α, and [ a ] = { x ∈ A : α ⁡ ( x ) = α ⁡ ( a ) } for each a∈A.

Solution 4.

  1. Reflexive: A mapping is a rule that assigns to every element a∈A exactly one element α ⁡ ( a ) ∈ B , so α ⁡ ( a ) = α ⁡ ( a ) for each a∈A. Hence a≡a .
  2. Symmetric: If a ≡ a1 , then α ⁡ ( a ) = α ⁡ ( a1 ) , so α ⁡ ( a1 ) = α ⁡ ( a ) . Hence a1 ≡ a .
  3. Transitive: Let a , a1 , and a2 ∈ A . If a ≡ a1 and a1 ≡ a2 , then α ⁡ ( a ) = α ⁡ ( a1 ) = α ⁡ ( a2 ) . Hence a ≡ a2 .
