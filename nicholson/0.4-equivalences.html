0.4 Equivalences

Relations

If A is a set, a subset ∼ of A×A is called a relation on A. For elements a and b in A, we customarily write:

  • a∼b to mean ( a , b ) ∈ ∼
  • a≁b to mean ( a , b ) ∉ ∼

Equivalences

A relation ≡ on a set A is called an equivalence on A if it satisfies the following conditions, where a, b, and c ∈A :

condition property
(1) for all a∈A , a≡a reflexive
(2) if a≡b , then b≡a symmetric
(3) if a≡b and b≡c , then a≡c transitive

If ≡ is an equivalence on a set A, the statement a≡b is read as “a is equivalent to b.”

Equivalnece Classes

Let ≡ be an equivalence on a set A. Given a∈A, the equivalence class [ a ] of a is defined as the set of all elements of A that are equivalent to a:

Examples 1–5 illustrate equivalences.

Example 1. Equality is an equivalence on any set A. If a∈A, the equivalence class of a is [ a ] = { x∈A | x=a } = { a } , the singleton.

Example 2. Being parallel is an equivalence on the set of lines in the plane. The equivalence class of a given line consists of all lines parallel to it.

Example 3. If X and Y are subsets of a finite set U, write X≡Y to mean | X | = | Y | . Then ≡ is an equivalence on the set of subsets of U, and [ X ] consists of all subsets with the same number of elements as X.

Example 4. Let α : A → B be a mapping. If a and a1 are elements of A, write a ≡ a1 to mean α ⁡ ( a ) = α ⁡ ( a 1 ) . Then ≡ is an equivalence on A, called the kernel equivalence of α, and [ a ] = { x ∈ A | α ⁡ ( x ) = α ⁡ ( a ) } for each a∈A.

Example 5. If m and n are integers, define m≡n to mean that m−n is even. Then ≡ is an equivalence on ℤ. In this case,

Theorem 1

Let ≡ be an equivalence on a set A and let a and b denote elements of A. Then

  1. a ∈ [ a ] for every a∈A
  2. [ a ] = [ b ] if and only if a≡b
  3. If a ∈ [ b ] then [ a ] = [ b ]
  4. If [ a ] ≠ [ b ] then [ a ] ∩ [ b ] = ∅

Proof.

  1. This is clear because a≡a for all a∈A by the reflexive property.
  2. If [ a ] = [ b ] , then a ∈ [ b ] by (1), so a≡b . Conversely, assume a≡b . If x ∈ [ a ] , then x≡a , so, since a≡b , we have x≡b by transitivity. Thus, x ∈ [ b ] and we have proved that [ a ] ⊆ [ b ] . The other inclusion [ b ] ⊆ [ a ] follows in the same way because b≡a by symmetry. Hence [ a ] = [ b ] .
  3. If a ∈ [ b ] , then a≡b , so [ a ] = [ b ] by (2).
  4. We argue by contradiction. If [ a ] ≠ [ b ] , we assume on the contrary that [ a ] ∩ [ b ] ≠ ∅ , say x ∈ [ a ] ∩ [ b ] . Then x≡a and x≡b , so a≡b by the symmetric and transitive properties. But then [ a ] = [ b ] by (2), a contradiction.

Summary – Equivalences

  • A relation on a set A is a subset of A×A.
  • An equivalence on a set A is a relation ∼ on A that is:
    • reflexive: ∀ a∈A , a∼a
    • symmetric: a∼b ⇒ b∼a
    • transitive: ( a∼b ) ∧ ( b∼c ) ⇒ a∼c
    and often written as ≡ rather than ∼
  • An equivalence class on a set A is a set defined as:
    • [ a ] = { x∈A | x≡a }
    then
    • ∀ a∈A , a∈ [ a ]
    • [ a ] = [ b ] ⇔ a≡b
    • a∈ [ b ] ⇒ [ a ] = [ b ]
    • [ a ] ≠ [ b ] ⇒ [ a ] ∩ [ b ] = ∅

Partitions

Disjoint

  • Two sets X and Y are called disjoint if X∩Y = ∅
  • a family of sets is called pairwise disjoint if any two (distinct) sets in the family are disjoint.

Partitions

If A is a nonempty set, a family 𝒫 of subsets of A is called a partition of A if

  1. no cell is empty,
  2. the cells are pairwise disjoint, and
  3. every element of A belongs to some cell.

The sets in 𝒫 are called the cells of the partition.

Example 6

The set A = { 1 , 2 , 3 } has five partitions:

Theorem 2 – Partition Theorem

If ≡ is any equivalence on a nonempty set A, the family of all equivalence classes is a partition of A that affords ≡.

Proof. The equivalence classes are nonempty and pairwise disjoint by (1) and (4) of Theorem 1, and every element of A belongs to some class (the one it generates). Hence, the equivalence classes are the cells of a partition.

To show that this partition affords ≡, it is enough to show that two elements a and b are equivalent if and only if they belong to the same equivalence class.

  1. If a≡b , then [ a ] = [ b ] by (2) of Theorem 1, so a and b belong to this common class.
  2. Conversely, if a and b belong to class [ c ] , then [ a ] = [ c ] = [ b ] by (3) of Theorem 1, so a≡b .

Partitions give rise to equivalence

If 𝒫 is a partition of the nonempty set A, and if a and b are elements of A, we define a∼b to mean that a and b are in the same cell of 𝒫. Then

  • because each a∈A lies in some cell, so a∼a (reflexive)
  • if a∼b, then a and b are in the same cell, so obviously b∼a (symmetric)
  • let a∼b and b∼c. Because b lies in a unique cell, a and c are in that same cell; that is, a∼c (transitive)
Hence, ∼ is an equivalence on A, and we say that it is the equivalence afforded by the partition.

Quotient Sets

If ≡ is an equivalence on A, the set of all equivalence classes is called the quotient set and denoted A≡.
