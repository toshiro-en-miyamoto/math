0.4 Equivalences

Relations

If A is a set, a subset ∼ of A×A is called a relation on A. For elements a and b in A, we customarily write:

  • a∼b to mean ab ∈∼
  • a≁b to mean ab ∉∼

Equivalences

A relation ≡ on a set A is called an equivalence on A if it satisfies the following conditions, where a, b, and c ∈A :

condition property
(1) for all a∈A , a≡a reflexive
(2) if a≡b , then b≡a symmetric
(3) if a≡b and b≡c , then a≡c transitive

If ≡ is an equivalence on a set A, the statement a≡b is read as “a is equivalent to b.”

Example 1. Equality is an equivalence on any set A. If a∈A, the equivalence class of a is [ a ] = { x∈A | x=a } = { a } , the singleton.

Solution 1.

  1. Reflexive: aa ∈≡ because for all a∈A, a=a .
  2. Symmetric: If a≡b , then a=b , which means b=a , hence b≡a .
  3. Transitive: If a≡b and b≡c , then a =b =c , which means a=c , hence a≡c .
To show that the equivalnece class of a is the singleton, suppose the class has two elements as [ a ] = { a , a1 } ; then we have a ≡ a1 meaning a = a1 , hence [ a ] = { a } , the singleton.

Example 2. Being parallel is an equivalence on the set of lines in the plane. The equivalence class of a given line consists of all lines parallel to it.

Solution 2.

  1. Reflexive: Let l be a line in the plane. The line is parallel to itself, hence l=l.
  2. Symmetric: Let l and l₁ be lines in the plane. If l∥l₁, then l₁∥l.
  3. Transitive: Let l, l₁, and l₂ be lines in the plane. If l∥l₁ and l₁∥l₂, then apparently l∥l₂.

Example 3. If X and Y are subsets of a finite set U, write X≡Y to mean | X | = | Y | . Then ≡ is an equivalence on the set of subsets of U, and | X | consists of all subsets with the same number of elements as X.

Solution 3.

  1. Reflexive: Let X be a subset of U. Apparently | X | = | X | .
  2. Symmetric: Let X and Y be subsets of U. If | X | = | Y | , which means X and Y have the same number of elements, hence | Y | = | X | .
  3. Transitive: Let X, Y, and Z be subsets of U. If | X | = | Y | and | Y | = | Z | , which means X, Y, and Z have the same number of elements, hence | X | = | Z | .
To prove that | X | consists of all subsets with the same number of elements as X, let | X | = n , then [ X ] = { A is a subset of U | A ≡ X } = { A is a subset of U | A has n elements } , which means all subsets with the same number of elements as X.
