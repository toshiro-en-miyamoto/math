1.4 Permutations

Permutations: Given an integer n ≥ 1 , write X n = { 1 , 2 , … , n } . A permutation of X n is a bijection σ : X n → X n .

Let S n be the set of all permutations of X n . Two permutations σ and τ in S n are equal if they are equal as functions, that is, if σ ⁡ ( k ) = τ ⁡ ( k ) for all k in X n .

Theorem 1. The set S n of permutations of X n has | S n | = n ! elements.

Proof. To construct a permutation, we must choose the numbers σ 1 , σ 2 , … , σ n from X n so that they are all distinct. Hence, we have n choices for σ 1 , then n − 1 choices for σ 2 , then n − 2 choices for σ 3 , and so on. Thus, σ can be chosen in n ⁢ ( n − 1 ) ⁢ ( n − 2 ) ⁢ … ⁢ 3 · 2 · 1 = n ! ways.

The parity: Two integers m and n are said to have the same parity if m ≡ n ⁡ ( mod ⁡ 2 ) ; equivalently, if they are both even or both odd.

Alternating groups: The set A n of all even permutations in S n is denoted A n . It is called the alternating group of degree n.

The alternating group of degree n plays an important role in the theory of groups (in Chapter 2).

Theorem 8. If n ≥ 2 , the alternating group A n has the following properties:

  1. ε ∈ A n
  2. if σ is in A n , then σ − 1 is in A n
  3. if σ and τ are in A n , then σ ∘ τ is in A n
  4. | A n | = ½ ⁢ n !

A set of permutations is called a group if it contains

  • the identity permutation,
  • the product of any two of its members, and
  • the inverse of any member.

Hence, S n is a group, and the first part of Theorem 8 shows that A n is a group.
