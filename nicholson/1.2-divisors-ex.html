Let n ∈ ℤ and d ∈ ℕ with d > 0 . Then there exist uniquely determined integers q and r such that

Proof. Let X = { n − t ⁢ d | t∈ℤ , n − t ⁢ d ≥ 0 } . First we show that X is nonempty.

  • If n≥0 , then for t = 0 , n − 0 ⁢ d = n is in X.
  • If n<0 , then for t = n , n − n ⁢ d = n ⁢ ( 1 − d ) is in X (because 1 ≤ d ).

Hence X is nonempty. Since X ⊆ ℕ , by the well-ordering principle, we know that X has a smallest member; call it r. Then r = n − q ⁢ d for some q, and r≥0 (because r ∈ X ). So it remains to show that r<d .

  • If r ≥ d , then 0 ≤ r − d = ( n − q ⁢ d ) − d = n − ( q + 1 ) ⁢ d .
  • This means that r − d is in X, contradicting the minimality of r.

This result proves the existence of q and r such that n = q ⁢ d + r and 0 ≤ r < d .

To complete the proof, we must show that q and r found above are the only members with these properties. To prove uniqueness, suppose also that n = q′ ⁢ d + r′ with 0 ≤ r′ < d . Assume r ≤ r′ (the case r′ ≤ r is similar). Then ( q − q′ ) ⁢ d = r′ − r is a nonnegative, integral multiple of d that is less than d (because r′ − r ≤ r′ < d ). This can occur only if r = r′ , which implies that q = q′ (because ( q − q′ ) ⁢ d = 0 and d > 0 ) and so proves uniqueness. ▮

Greatest Common Divisors

Let m and n be integers. An integer d is called a greatest common divisor of m and n if:

  1. d ≥ 1
  2. ( d ∣ m ) ∧ ( d ∣ n )
  3. ( k ∣ m ) ∧ ( k ∣ n ) ⇒ k ∣ d

When it exists we write d = gcd ⁡ ( m , n ) .
