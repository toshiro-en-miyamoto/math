0.3 Mappings

In this section, we introduce functions from any set A to any set B. These more general functions are called mappings to avoid confusion. In this generality, sets and mappings are the language of abstract algebra.

Mappings

In general, if A and B are sets, a mapping (or function) α from A to B, written

is a rule that assigns to every element a of A exactly one element α ⁡ ( a ) of B.

  • This assignment is sometimes denoted a ↦ α ⁡ ( a )
  • We refer to A and B as the domain and codomain, respectively, of the mapping α.
  • For a∈A, the unique element b∈B such that b = α ⁡ ( a ) is called the image of a under α.

The process of defining a mapping α consists of two parts:

  1. we must specify the domain A and codomain B of α,
  2. for every a∈A, we must specify exactly one element α ⁡ ( a ) ∈ B that α assigns to a.
We refer to this latter task as defining the action of α and then say that the mapping is well defined.

If the domain and codomain are sets of numbers, the most common way to define a mapping is by means of a formula. Thus, α ⁡ ( x ) = x 2 + 1 and β ⁡ ( x ) = 3 ⁢ x − 2 define mappings ℝ → ℝ .

Proof.

Theorem 3 – Distributive, one-to-one, and onto.

Let A → α B → β C → γ D be mappings. Then

  1. γ ∘ ( β ∘ α ) = ( γ ∘ β ) ∘ α
  2. If α and β are both one-to-one, the same is true of β ∘ α
  3. If α and β are both onto, the same is true of β ∘ α

Proof.

  1. If a∈A , then [ γ ∘ ( β ∘ α ) ] ⁡ ( a ) = γ ∘ [ β ∘ α ⁡ ( a ) ] = γ ⁡ [ β ⁡ ( α ⁡ ( a ) ) ] = γ ∘ β ⁡ [ α ⁡ ( a ) ] = [ ( γ ∘ β ) ∘ α ] ⁡ ( a ) .
  2. If α and β are one-to-one, suppose that β ∘ α ⁡ ( a ) = β ∘ α ⁡ ( a1 ) , where a , a1 ∈A . Thus, β ⁡ [ α ⁡ ( a ) ] = β ⁡ [ α ⁡ ( a1 ) ] , so α ⁡ ( a ) = α ⁡ ( a1 ) because β is one-to-one. But then a = a 1 because α is one-to-one. This shows that β ∘ α is one-to-one.
  3. Now assume that α and β are both onto. If c∈C, we have c = β ⁡ ( b ) for some b∈B (because β is onto) and then b = α ⁡ ( a ) for some a∈A (because α is onto). Hence, c = β ⁡ [ α ⁡ ( a ) ] = β ∘ α ⁡ ( a ) , proving that β ∘ α is onto.

A mapping α : A → B that has an inverse is called an invertible mapping, and the inverse mapping is denoted

α-1 is the unique mapping satisfying
  • α-1 ∘ α = 1A , and
  • α ∘ α-1 = 1B
or in other words
  • for all a∈A, α-1 ⁡ [ α ⁡ ( a ) ] = a , and
  • for all b∈B, α ⁡ [ α-1 ⁡ ( b ) ] = b

These are the Fundamental Identities relating α and α-1, and they show that the action of each of α and α-1 undoes the action of the other.

Theorem 5

Let α : A → B and β : B → C denote mappings.

  1. 1A : A → A is invertible and 1 A -1 = 1A
  2. If α is invertible, then α-1 is invertible and ( α -1 ) -1 = α .
  3. If α and β are both invertible, then β ∘ α is invertible and ( β ∘ α ) -1 = α -1 ∘ β -1 .

Proof.

  1. This result follows because 1A ∘ 1A = 1A .
  2. We have α-1 ∘ α = 1A and α ∘ α-1 = 1B , so α is the inverse of α-1.
  3. Compute ( β ∘ α ) ∘ ( α-1 ∘ β-1 ) = β ∘ ( α ∘ α-1 ) ∘ β-1 = β ∘ 1B ∘ β-1 = β ∘ β-1 = 1C . A similar calculation shows that ( α-1 ∘ β-1 ) ∘ ( β ∘ α ) = 1A , so α-1 ∘ β-1 is the inverse of β ∘ α . Note the order of the factors.

Theorem 6 – Invertibility Theorem

A mapping α : A → B is invertible if and only if it is both one-to-one and onto (that is, α is a bijection).

Proof.

  • Assume α-1 exists. If α ⁡ ( a ) = α ⁡ ( a1 ) , then a = α-1 ⁡ [ α ⁡ ( a ) ] = α-1 ⁡ [ α ⁡ ( a1 ) ] = a1 by one of the fundamental identities. Hence, α is one-to-one. If b∈B, then b = α ⁡ [ α-1 ⁡ ( b ) ] by the other fundamental identity, and so α is onto.
  • Conversely, assume α is onto and one-to-one. Given b∈B, there exists a∈A such that α ⁡ ( a ) = b (because α is onto) and a is unique (because α is one-to-one, verify). Hence, we may define β : B → A by β ⁡ ( b ) = a where a is the unique element of A with α ⁡ ( a ) = b . Thus, α ∘ β ⁡ ( b ) = α ⁡ ( a ) = b for each b∈B, so α ∘ β = 1B . If a∈A, write α ⁡ ( a ) = b . Hence, β ⁡ ( b ) = a by the definition of β, so β ∘ α ⁡ ( a ) = β ⁡ ( b ) = a . This means that β ∘ α = 1A , so β is the inverse of α.
