0.1 Proofs

Implies

Meaning
If p is true, then q is true.
Read it as
p implies q.

Here, p is the hypothesis and q the conclusion of the implication.

Method of Direct Proof

To prove p ⇒ q , demonstrate directly that q is true whenever p is true.

Example 1. If n is an odd integer, show that n2 is odd.

Solution. If n is odd, it has the form n = 2⁢k + 1 for some integer k. Then n2 = 4⁢k2 + 4⁢k + 1 = 2⁢ ( 2⁢k2 + 2⁢k ) + 1 is also odd because 2⁢k2 + 2⁢k is an integer.

Method to Reduction to Cases

Suppose that you are asked to verify that x2 ≥ 0 for every integer n. This expression is an implication: If x is an integer, then x2 ≥ 0 . To prove it, you might consider separately the cases that n>0, n=0, and n<0 and then show that x2 ≥ 0 in each case.

To prove p ⇒ q , show that p implies at least one of a list p1 , p2 , … , pn of statements (the cases) and that pi ⇒ q for each i.

Method of Proof by Contradiction

The method is a formal version of a debating strategy whereby the debater assumes the truth of an opponent's position and shows that it leads to an absurd conclusion.

To prove p ⇒ q , show that the assumption that both p is true and q is false leads to a contradiction.

Counterexamples

Note that if you can find even one example for which an implication is not valid (counterexample), the implication is false. Thus, disproving implications in a sense is easier than proving them.

Logical Equivalence

Meaning
If both p ⇒ q and q ⇒ p are valid, the statements p and q are called logically equivalent.
Read it as
p if and only if q.
